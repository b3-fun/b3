## Primeros Pasos

### Prerrequisitos

Antes de comenzar a entrenar tu agente, asegúrate de tener:

- **Unity 2022.3 LTS** o más reciente
- **Python 3.8+** con pip instalado
- **ML-Agents Toolkit** de Unity
- **PyTorch** o **TensorFlow** para el entrenamiento

### Pasos de Instalación

<Steps>
  <Step title="Instalar Unity ML-Agents">
    ```bash
    pip install mlagents
    ```
  </Step>
  <Step title="Clonar la Plantilla de Entrenamiento HypeDuel">
    ```bash
    git clone https://github.com/hypeduel/agent-training-template
    cd agent-training-template
    ```
  </Step>
  <Step title="Instalar Dependencias">
    ```bash
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Abrir en Unity">
    Abre el proyecto en Unity Hub y carga la escena de entrenamiento
  </Step>
</Steps>

## Configuración del Entrenamiento

### Parámetros de Comportamiento del Agente

Configura los parámetros de aprendizaje de tu agente en el archivo `config.yaml`:

```yaml
behaviors:
  HypeDuelAgent:
    trainer_type: ppo
    hyperparameters:
      batch_size: 1024
      buffer_size: 10240
      learning_rate: 3.0e-4
      beta: 5.0e-4
      epsilon: 0.2
      lambd: 0.99
      num_epoch: 3
      learning_rate_schedule: linear
    network_settings:
      normalize: false
      hidden_units: 128
      num_layers: 2
    reward_signals:
      extrinsic:
        gamma: 0.99
        strength: 1.0
    max_steps: 500000
    time_horizon: 64
    summary_freq: 10000
```

### Espacio de Observación

Tu agente recibirá observaciones que incluyen:

- **Estado de la Arena**: Posiciones actuales, salud y potenciadores
- **Comportamiento del Oponente**: Patrones de movimiento y acciones recientes
- **Datos Ambientales**: Límites de la arena, obstáculos y peligros
- **Estado del Juego**: Tiempo restante, diferencia de puntuación y disponibilidad de potenciadores

### Espacio de Acción

Los agentes pueden realizar acciones como:

- **Movimiento**: Adelante, atrás, izquierda, derecha
- **Combate**: Atacar, defender, habilidades especiales
- **Estrategia**: Posicionamiento, recolección de potenciadores, maniobras defensivas

## Proceso de Entrenamiento

### Entrenamiento Local

Inicia el entrenamiento de tu agente localmente:

```bash
mlagents-learn config.yaml --run-id=hypeduel-agent-v1
```

### Envío a la Arena

Una vez que tu agente esté entrenado:

1. **Prueba de Rendimiento**: Valida contra oponentes de referencia
2. **Optimización del Modelo**: Asegura velocidad de inferencia eficiente
3. **Envío a HypeDuel**: Sube tu archivo de modelo `.onnx`
4. **Registro en la Arena**: Registra tu agente para arenas específicas

## Mejores Prácticas

### Consejos de Entrenamiento

- **Comienza Simple**: Empieza con comportamientos básicos antes de estrategias complejas
- **Ingeniería de Recompensas**: Diseña recompensas que fomenten comportamientos deseados
- **Evaluación Regular**: Prueba contra oponentes diversos frecuentemente
- **Complejidad Incremental**: Introduce mecánicas avanzadas gradualmente

### Optimización del Rendimiento

- **Normalización de Observaciones**: Escala las entradas para un entrenamiento estable
- **Diseño del Espacio de Acción**: Equilibra la expresividad con la eficiencia de entrenamiento
- **Gestión de Memoria**: Optimiza tanto para entrenamiento como para inferencia
- **Compresión de Modelo**: Reduce el tamaño del modelo para un despliegue más rápido en la arena

## Solución de Problemas

### Problemas Comunes

<Accordion title="Entrenamiento No Convergente">
  Verifica la tasa de aprendizaje, escalado de recompensas y normalización de observaciones. Considera reducir la complejidad o ajustar los hiperparámetros.
</Accordion>

<Accordion title="Rendimiento del Agente Estancado">
  Implementa aprendizaje por currículum, aumenta la diversidad del entorno o prueba el entrenamiento de autojugadores contra oponentes más fuertes.
</Accordion>

<Accordion title="Inferencia Demasiado Lenta">
  Optimiza la arquitectura del modelo, usa el tiempo de ejecución de ONNX o implementa el almacenamiento en caché de acciones para decisiones repetidas.
</Accordion>
