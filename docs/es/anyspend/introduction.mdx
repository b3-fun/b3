## ¿Qué es AnySpend?

AnySpend es la infraestructura de pagos de B3 para cripto. Maneja el enrutamiento entre cadenas, la conversión de tokens y rampas de acceso a fiat para que tus usuarios puedan pagar con lo que tengan en su billetera. Recibes el token que deseas, en la cadena que deseas.

[Prueba AnySpend en tu navegador](https://anyspend.com/swap) | [Demostración del SDK (Widget Lab)](https://sdk-demo.anyspend.com/)

## Cómo funciona

<Steps>
<Step title="El usuario inicia el pago">
El usuario hace clic en un botón de pago o interactúa con tu UI, eligiendo su token preferido.
</Step>

<Step title="AnySpend cotiza una ruta">
El motor de enrutamiento encuentra la mejor ruta para convertir su token al token de destino.
</Step>

<Step title="Ejecución entre cadenas">
Si es necesario, los tokens se transfieren automáticamente entre cadenas.
</Step>

<Step title="Entrega">
Los tokens llegan a la dirección de destino, ejecutan cualquier lógica personalizada y activan tus callbacks de éxito.
</Step>
</Steps>

## Inicio rápido

```tsx
import { AnySpend, AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";

import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production" partnerId="your-awesome-partner-id">
      <AnyspendProvider>
        <AnySpend />
      </AnyspendProvider>
    </B3Provider>
  );
}

export default App
```
