## ¿Qué es AnySpend?

AnySpend es la infraestructura de pagos integral de B3 que permite transacciones cripto sin interrupciones a través de múltiples blockchains. Ya sea que estés construyendo una aplicación DeFi, un mercado de NFT o una plataforma de juegos, AnySpend proporciona los componentes y las API que necesitas para aceptar pagos en cualquier token.

[Prueba AnySpend en tu navegador](https://anyspend.com/swap)

## Características

<Steps>
<Step title="El Usuario Inicia el Pago">
El usuario hace clic en un botón de pago o interactúa con tu interfaz, eligiendo su token preferido
</Step>

<Step title="Cotizaciones de AnySpend">
Nuestro motor de enrutamiento encuentra la mejor ruta para convertir su token al token de destino requerido
</Step>

<Step title="Ejecución Cross-Chain">
Si es necesario, los tokens se transfieren a través de cadenas usando infraestructura segura y probada en batalla
</Step>

<Step title="Entrega Final">
Los tokens llegan a la dirección de destino, ejecutan cualquier lógica personalizada y activan tus callbacks de éxito
</Step>
</Steps>

## Inicio Rápido

Comienza con AnySpend en solo unos pocos pasos:

```tsx
import { AnySpend, AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";

import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production">
      <AnyspendProvider>
        <AnySpend />
      </AnyspendProvider>
    </B3Provider>
  );
}

export default App
```
