## Componentes principales

### `<AnySpend>`

El componente de interfaz principal para intercambios de token a token, rampas de entrada fiat y intercambios entre cadenas.

```tsx title="Uso Básico" icon="react"
import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

<AnySpend
  mode="modal"
  defaultActiveTab="crypto"
  destinationTokenAddress="0x..."
  destinationTokenChainId={8453}
  recipientAddress="0x..."
/>;
```

#### Props

<ParamField path="mode" type="'modal' | 'page'" default="'modal'">
  Mostrar como superposición modal o página completa
</ParamField>

<ParamField path="defaultActiveTab" type="'crypto' | 'fiat'" default="'crypto'">
  Pestaña de método de pago inicial
</ParamField>

<ParamField path="destinationTokenAddress" type="string">
  Dirección del contrato del token objetivo
</ParamField>

<ParamField path="destinationTokenChainId" type="number">
  ID de la cadena del token destino
</ParamField>

<ParamField path="recipientAddress" type="string">
  Dirección de la billetera del destinatario
</ParamField>

<ParamField path="sourceChainId" type="number">
  Preseleccionar la cadena de origen
</ParamField>

<ParamField path="loadOrder" type="string">
  Cargar y mostrar una orden existente por ID
</ParamField>

<ParamField path="hideTransactionHistoryButton" type="boolean" default="false">
  Ocultar el botón de historial de transacciones
</ParamField>

<ParamField path="hideHeader" type="boolean">
  Ocultar el encabezado del componente
</ParamField>

<ParamField path="hideBottomNavigation" type="boolean">
  Ocultar la barra de navegación inferior
</ParamField>

<ParamField path="onTokenSelect" type="(token: Token, event: { preventDefault: () => void }) => void">
  Llamado cuando se selecciona un token. Llamar a `event.preventDefault()` para prevenir el comportamiento de selección de token predeterminado.
</ParamField>

<ParamField path="onSuccess" type="(txHash?: string) => void">
  Callback disparado en transacción exitosa
</ParamField>

<ParamField path="customUsdInputValues" type="string[]">
  Presets de entrada fiat personalizados (p. ej., `["10", "25", "50", "100"]`)
</ParamField>

<ParamField path="disableUrlParamManagement" type="boolean">
  Deshabilitar la sincronización de parámetros de URL para la configuración de intercambio
</ParamField>

<ParamField path="allowDirectTransfer" type="boolean">
  Habilitar transferencias directas cuando el token/cadena de origen y destino coinciden (omite el backend de intercambio)
</ParamField>

<ParamField path="destinationTokenAmount" type="string">
  Cantidad de destino fija en wei. Cuando se establece, los usuarios no pueden cambiar la cantidad.
</ParamField>

<ParamField path="senderAddress" type="string">
  Prellenar la dirección del remitente para mostrar saldos de tokens antes de la conexión de la billetera
</ParamField>

<ParamField path="callbackMetadata" type="Record<string, unknown>">
  Metadatos opacos pasados a los callbacks de orden (p. ej., datos de formulario de flujo de trabajo)
</ParamField>

<ParamField path="returnToHomeUrl" type="string">
  URL para redirigir desde el botón "Volver al inicio" en la pantalla de finalización
</ParamField>

<ParamField path="customRecipientLabel" type="string">
  Etiqueta personalizada para la visualización del destinatario (p. ej., "Bot de Telegram OBSN")
</ParamField>

<ParamField path="returnHomeLabel" type="string">
  Etiqueta personalizada para el botón de retorno al inicio
</ParamField>

<ParamField path="classes" type="AnySpendClasses">
  Sobrescrituras de clases CSS para elementos específicos. Ver [Personalización](/anyspend/customization#css-class-overrides).
</ParamField>

<ParamField path="slots" type="AnySpendSlots">
  Sobrescrituras de funciones de renderizado para elementos de UI reemplazables. Ver [Personalización](/anyspend/customization#slots).
</ParamField>

<ParamField path="content" type="AnySpendContent">
  Sobrescrituras de texto/mensaje para estados de éxito, error y procesamiento. Ver [Personalización](/anyspend/customization#content).
</ParamField>

<ParamField path="theme" type="AnySpendTheme">
  Configuración de color y tema. Ver [Personalización](/anyspend/customization#theme).
</ParamField>

### `<AnySpendDeposit>`

Una interfaz de depósito simplificada que admite la selección de cadena, pagos en cripto/fiat y depósitos en contratos personalizados.

```tsx title="Uso Básico" icon="arrow-down"
import { AnySpendDeposit } from "@b3dotfun/sdk/anyspend/react";

<AnySpendDeposit
  recipientAddress="0x..."
  destinationTokenAddress="0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913"
  destinationTokenChainId={8453}
  onSuccess={(amount) => console.log("Depositado:", amount)}
/>;
```

<ParamField path="recipientAddress" type="string" required>
  La dirección de la billetera para recibir el depósito

</ParamField>

#### DepositContractConfig

Usa esto para configurar depósitos en contratos personalizados:

```typescript title="Definición de Tipo" icon="code"
interface DepositContractConfig {
  /** Cadena JSON de ABI de función personalizada */
  functionAbi: string;
  /** El nombre de la función a llamar */
  functionName: string;
  /** Argumentos para la función. Usa "{{amount_out}}" como marcador de posición para la cantidad del depósito */
  functionArgs: string[];
  /** Dirección del contrato para depositar */
  to: string;
  /** Dirección del gastador opcional si es diferente del contrato */
  spenderAddress?: string;
  /** Etiqueta de acción personalizada (p. ej., "Apostar", "Depositar") */
  action?: string;
}
```

#### ChainConfig

```typescript title="Definición de Tipo" icon="code"
interface ChainConfig {
  id: number;       // ID de la cadena
  name: string;     // Nombre para mostrar
  iconUrl?: string; // URL del icono opcional
}
```

### `<AnySpendCheckout>`

Un componente de checkout completo con visualización del carrito, resumen del pedido, formularios personalizados, selección de envío, códigos de descuento y soporte para múltiples métodos de pago.

```tsx title="Uso Básico" icon="cart-shopping"
import { AnySpendCheckout } from "@b3dotfun/sdk/anyspend/react";

<AnySpendCheckout
  recipientAddress="0x..."
  destinationTokenAddress="0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913"
  destinationTokenChainId={8453}
  items={[
    { name: "Plan Pro", amount: "10000000", quantity: 1 },
  ]}
  organizationName="Acme Inc"
  onSuccess={(result) => console.log("Pagado:", result)}
/>;
```

Para la documentación completa del checkout, ver la [Guía de Checkout](/anyspend/checkout).

#### Props

<ParamField path="recipientAddress" type="string" required>
  Dirección de la billetera del destinatario del pago
</ParamField>

<ParamField path="destinationTokenAddress" type="string" required>
  Dirección del contrato del token de destino
</ParamField>

<ParamField path="destinationTokenChainId"
