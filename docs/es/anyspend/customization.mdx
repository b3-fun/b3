## Slots

Los slots te permiten reemplazar secciones enteras de la interfaz de usuario con tus propios componentes React. Cada slot recibe propiedades de AnySpend que puedes usar en tu implementación personalizada.

```typescript title="Interfaz AnySpendSlots" icon="code"
interface AnySpendSlots {
  /** Reemplaza el botón principal de acción/pago */
  actionButton?: (props: ActionButtonSlotProps) => ReactNode;
  /** Reemplaza el botón de conectar cartera */
  connectWalletButton?: (props: ConnectWalletButtonSlotProps) => ReactNode;
  /** Reemplaza el encabezado */
  header?: (props: { mode: "page" | "modal" }) => ReactNode;
  /** Reemplaza el pie de página */
  footer?: ReactNode;
  /** Reemplaza la pantalla de éxito */
  successScreen?: (props: SuccessScreenSlotProps) => ReactNode;
  /** Reemplaza la pantalla de error */
  errorScreen?: (props: ErrorScreenSlotProps) => ReactNode;
  /** Reemplaza el panel del formulario de pago (solo en checkout) */
  checkoutForm?: (props: CheckoutFormComponentProps) => ReactNode;
  /** Reemplaza el selector de método de envío (solo en checkout) */
  shippingSelector?: (props: ShippingSelectorSlotProps) => ReactNode;
  /** Reemplaza la entrada de código de descuento (solo en checkout) */
  discountInput?: (props: DiscountInputSlotProps) => ReactNode;
}
```

<Tip>
  Los slots `checkoutForm`, `shippingSelector` y `discountInput` solo se usan en `<AnySpendCheckout>` y `<AnySpendCheckoutTrigger>`. Consulta la [Guía de Checkout](/anyspend/checkout) para detalles sobre las características de formulario/envío/descuento.
</Tip>

## Tema

Configura la apariencia visual de los componentes AnySpend con colores de marca.

```typescript title="Interfaz AnySpendTheme" icon="code"
interface AnySpendTheme {
  /** Color principal de la marca (hex). Aplicado como el color de acento principal. */
  brandColor?: string;
  /** Sobrescrituras individuales de colores (valores hex) */
  colors?: Partial<{
    primary: string;         // Color principal de texto
    secondary: string;       // Color secundario de texto
    tertiary: string;        // Color de texto atenuado
    surfacePrimary: string;  // Color de fondo primario
    surfaceSecondary: string; // Fondo secundario (tarjetas, paneles)
    brand: string;           // Color de acento/marca (botones, enlaces)
    borderPrimary: string;   // Color de borde primario
    borderSecondary: string; // Color de borde secundario
  }>;
}
```

<Note>
  La propiedad `brandColor` es un atajo que establece el color `brand`. Si se proporcionan tanto `brandColor` como `colors.brand`, `colors.brand` tiene prioridad.
</Note>

### Ejemplos de tema

<Tabs>
<Tab title="Solo Color de Marca">

</Tab>

```tsx
    },
  }}
```
