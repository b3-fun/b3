## Envío de Artículos

Envía ideas de artículos a sitios de contenido cripto. La IA valida tu tema y genera títulos y descripciones optimizados.

### Cómo Funciona

<Steps>
  <Step title="Enviar Tema o Tweet">
    Proporciona una URL de tweet, idea de tema o título completo del artículo
  </Step>
  <Step title="Validación por IA (Gratis)">
    La IA puntúa tu contenido (0-100) y genera sugerencias si es necesario
  </Step>
  <Step title="Obtener Aprobación">
    - **Puntuación ≥80:** Tu entrada es perfecta, aceptada tal cual
    - **Puntuación 70-79:** Aprobada con sugerencias mejoradas por IA
    - **Puntuación &lt;70:** Rechazada - intenta con un tema diferente
  </Step>
  <Step title="Enviar y Pagar">
    Una firma x402 para enviar tu solicitud de artículo
  </Step>
  <Step title="Publicación">
    El sitio publica tu artículo y recibes la URL en vivo
  </Step>
</Steps>

### Precios

Los precios varían según el sitio, típicamente **$3-50 USDC** por artículo. Cada sitio establece su propia tarifa basada en el tráfico y la audiencia.

<Note>
  ¡La validación es gratuita! Puedes validar tantas ideas como quieras antes de comprometerte al pago.
</Note>

### Compra de Espacio Publicitario (Pago x402)

```bash
POST https://x402.anyspend.com/ads/buy
```

Compra espacios publicitarios programáticamente usando pagos sin gas x402. Este endpoint requiere autorización de pago x402.

**Cuerpo de la Solicitud:**

<CodeGroup>
```json Request
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Texto alternativo",
    "title": "Título del anuncio",
    "description": "Descripción"
  }
}
```

```json Success Response (200)
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json Error Response (400)
{
  "success": false,
  "error": "Espacio ya reservado durante este tiempo"
}
```
</CodeGroup>

**Parámetros:**
- `productId` (requerido) - El ID del espacio publicitario de `/ads/slots`
- `buyerAddress` (requerido) - Tu dirección de cartera
- `startTime` (requerido) - Hora de inicio en formato ISO 8601
- `endTime` (requerido) - Hora de fin en formato ISO 8601
- `adContent.imageUrl` (requerido) - URL directa a la imagen del anuncio
- `adContent.linkUrl` (requerido) - URL de la página de destino
- `adContent.altText` (requerido) - Texto alternativo de la imagen
- `adContent.title` (opcional) - Título del anuncio
- `adContent.description` (opcional) - Descripción del anuncio

**Ejemplo en JavaScript/TypeScript:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// Crear cliente de cartera
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// Envolver fetch con pago x402
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // Máximo 1000 USDC
);

// Comprar espacio publicitario
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: 'Mi anuncio',
        title: '¡Conoce nuestro producto!',
        description: 'Oferta limitada'
      }
    })
  }
);

const result = await response.json();
console.log('ID de reserva:', result.data.bookingId);
```

<Note>
  Consulta la [documentación del SDK x402](/anyspend/x402-overview) para más detalles sobre el uso de `wrapFetchWithPayment`.
</Note>

## Envío de Artículos Endpoints

### Validar Idea de Artículo (Gratis)

```bash
POST https://x402.anyspend.com/articles/validate
```

Valida una idea de artículo y obtén un título y descripción generados por IA. Este endpoint es gratuito - no se requiere pago.

**Cuerpo de la Solicitud:**

<CodeGroup>
```json Request
{
  "siteDomain": "baseradar.com",
  "userInput": "Cómo Usar Base Pay para Creadores en Tu DApp"
}
```

```json Approved Response (Score ≥70)
{
  "success": true,
  "data": {
    "approved": true,
    "score": 92,
    "reasoning": "Excelente coincidencia de tema para el contenido del ecosistema Base",
    "estimatedPrice": "15000000",
    "suggestedTitle": "Cómo Usar Base Pay para Creadores en Tu DApp",
    "suggestedDescription": "Aprende cómo integrar Base Pay...",
    "userProvidedTitle": true,
    "message": "Tu entrada está aprobada tal cual"
  }
}
```

```json Rejected Response (Score <70)
{
  "success": false,
  "data": {
    "approved": false,
    "score": 45,
    "reasoning": "El tema no se alinea con el enfoque del sitio en el ecosistema Base",
    "estimatedPrice": "15000000",
    "message": "El contenido no se ajusta al enfoque del sitio"
  }
}
```
</CodeGroup>

**Parámetros:**
- `siteDomain` (requerido) - Dominio del sitio objetivo (por ejemplo, "baseradar.com", "solanaecho.com")
- `userInput` (requerido) - Puede ser:
  - Título completo del artículo (10-200 caracteres)
  - URL de Tweet (por ejemplo, "https://x.com/username/status/...")
  - Tema o idea (por ejemplo, "base pay para creadores")

**Puntuación de IA:**
- **Puntuación ≥
