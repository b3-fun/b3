## Preguntas Frecuentes

<AccordionGroup>
<Accordion title="¿Por qué no puedo iniciar sesión en mi cuenta?">
Asegúrate de estar utilizando el método de inicio de sesión correcto y que tu cartera o cuenta social esté conectada correctamente.
</Accordion>

<Accordion title="No recibí mi correo electrónico de verificación. ¿Qué debo hacer?">
Revisa la carpeta de spam o intenta reenviarlo. Si aún así no llega, intenta usar un método de inicio de sesión diferente como cartera o Discord, o contáctanos a través de nuestro portal de soporte para asistencia adicional.
</Accordion>

<Accordion title="¿Qué métodos de inicio de sesión son compatibles?">
Puedes iniciar sesión usando correo electrónico, Discord, Telegram, cartera o cuentas sociales como Google y Apple.
</Accordion>

<Accordion title="¿Puedo vincular múltiples cuentas a un perfil?">
Sí. Puedes conectar tu cartera, Discord y otras cuentas en la configuración de tu perfil en [**basement.fun**](https://basement.fun/).
</Accordion>
</AccordionGroup>

<Info>
Si continúas teniendo problemas para iniciar sesión, asegúrate de estar utilizando el mismo método de inicio de sesión que usaste originalmente para crear tu cuenta.
</Info>
