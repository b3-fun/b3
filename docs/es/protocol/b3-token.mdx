## ¿Qué es \$B3?

**\$B3** es el token de utilidad que alimenta todo el ecosistema B3. Como el token fundacional para una red multichain de escalado horizontal, B3 habilita la gobernanza, staking y valor a través de appchains y aplicaciones construidas en B3.
