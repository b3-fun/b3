## ¿Qué es \$B3?

**\$B3** es el token de utilidad que impulsa todo el ecosistema B3. Como el token fundamental para una red multichain de escalado horizontal, B3 habilita la gobernanza, el staking y el valor a través de appchains y aplicaciones construidas sobre B3.
