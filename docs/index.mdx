## About B3

**The Open Gaming Superchain**

B3 is a fun-first gaming ecosystem built on Base, designed to bring the next generation of gamers and game creators onchain. Created by a team of Base/Coinbase alumni and OG Ethereum contributors, B3 envisions a future where onchain is the new online.

<Info>
  B3 operates as an L3 that settles on Base, providing dedicated blockspace specifically for gaming with lightning-fast
  transactions and sub-cent fees.
</Info>

<Tabs>
<Tab title="Gaming Platforms">

function GameLauncher() {

</Tab>

<Tab title="NFT Marketplaces">
```tsx
import { AnySpendNFTButton } from "@b3dotfun/sdk/anyspend/react";

function NFTCard() {
  return (
    <AnySpendNFTButton
      nftContract={{
        chainId: {networks.b3.mainnet.chainId},
        contractAddress: "{examples.nftContract.contractAddress}",
        price: "{examples.nftContract.price}",
        currency: { symbol: "ETH", decimals: 18 }
      }}
      recipientAddress={userAddress}
      onSuccess={(txHash) => showSuccess(txHash)}
    />
  );
}
````

</Tab>
