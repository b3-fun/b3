### Βήμα 1: Εγκατάσταση του B3 SDK
<Steps>
  <Step title="Εγκατάσταση του πακέτου SDK">
    Επιλέξτε τον προτιμώμενο διαχειριστή πακέτων σας για να εγκαταστήσετε το B3 SDK:

    <CodeGroup>
    ```bash npm
    npm install @b3dotfun/sdk
    ```
    ```bash pnpm
    pnpm add @b3dotfun/sdk
    ```
    ```bash yarn
    yarn add @b3dotfun/sdk
    ```
    </CodeGroup>

    <Tip>Το B3 SDK περιλαμβάνει ορισμούς TypeScript εξ' ορισμού!</Tip>
  </Step>

  <Step title="Εισαγωγή CSS στυλ">
    Εισάγετε τα προεπιλεγμένα στυλ στην εφαρμογή σας:

    ```typescript
    import "@b3dotfun/sdk/index.css";
    ```
  </Step>
</Steps>

### Βήμα 2: Προσθήκη του B3Provider
Περικλείστε το B3Provider γύρω από τη ρίζα της εφαρμογής σας:

```tsx
import { B3Provider, SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

function App() {
  return (
    <B3Provider environment="production">
      {children}
    </B3Provider>
  );
}
```


### Βήμα 3: Χρήση του B3 SDK
<AccordionGroup>
 <Accordion icon="swap" title="Άμεσοι Λογαριασμοί">
    Ενεργοποιήστε την παγκόσμια ταυτοποίηση λογαριασμών με το B3:

    ```tsx
    import { SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

    function App() {
      return (
        <B3Provider environment="production">
          <SignInWithB3
            provider={{ strategy: "google" }}
            partnerId="your-partner-id"
            onLoginSuccess={(globalAccount) => {
              console.log("Authenticated:", globalAccount);
            }}
          />
        </B3Provider>
      );
    }
    ```
  </Accordion>
  <Accordion icon="swap" title="Πρόσβαση σε Ρευστότητα">
    Ενεργοποιήστε διασυναλλαγές cross-chain & προσαρμοσμένες αλληλεπιδράσεις με το AnySpend:

    ```tsx
    import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

    function SwapPage() {
      return <AnySpend mode="page" />;
    }
    ```
  </Accordion>
</AccordionGroup>
