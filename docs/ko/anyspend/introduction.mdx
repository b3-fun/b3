## AnySpend이란?

AnySpend은 여러 블록체인에서 원활한 암호화폐 거래를 가능하게 하는 B3의 포괄적인 결제 인프라입니다. DeFi 애플리케이션, NFT 마켓플레이스, 게이밍 플랫폼을 구축하든, AnySpend은 원하는 토큰으로 결제를 받을 수 있는 컴포넌트와 API를 제공합니다.

[브라우저에서 AnySpend 시도하기](https://anyspend.com/swap)

## 특징

<Steps>
<Step title="사용자 결제 시작">
사용자가 결제 버튼을 클릭하거나 인터페이스와 상호작용하여 선호하는 토큰을 선택합니다
</Step>

<Step title="AnySpend 견적">
라우팅 엔진이 해당 토큰을 필요한 목적지 토큰으로 변환하기 위한 최적의 경로를 찾습니다
</Step>

<Step title="크로스체인 실행">
필요한 경우, 토큰은 안전하고 검증된 인프라를 사용하여 체인 간에 브리징됩니다
</Step>

<Step title="최종 전달">
토큰이 목적지 주소에 도착하여 사용자 정의 로직을 실행하고 성공 콜백을 트리거합니다
</Step>
</Steps>

## 빠른 시작

단 몇 단계만으로 AnySpend를 시작하세요:

```tsx
import { AnySpend, AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";

import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production">
      <AnyspendProvider>
        <AnySpend />
      </AnyspendProvider>
    </B3Provider>
  );
}

export default App
```
