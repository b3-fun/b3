## 기능

사용자가 특정 지갑 주소로 토큰을 보낼 수 있는 호스팅된 입금 페이지입니다. 사용자는 지갑을 연결하고, 토큰을 선택하며, AnySpend는 필요한 경우 크로스 체인 라우팅을 처리합니다. URL을 생성하면 사용자가 클릭합니다.

## 작동 방식

1. **앱이 입금 URL을 생성**합니다. 수신자 주소, 체인, 토큰이 포함됩니다.
2. **사용자가 지갑을 연결**합니다 (MetaMask, Coinbase, Rainbow, Rabby, Trust Wallet).
3. **사용자가 토큰을 선택**합니다. 사용 가능한 잔액에서 선택합니다.
4. **AnySpend이 결제를 크로스 체인으로 라우팅**합니다. 필요한 경우 목적지 토큰으로 변환합니다.
5. **완료** -- 선택적으로 사용자를 앱으로 다시 리디렉션합니다.

## URL 파라미터

입금 페이지는 다음 URL 파라미터를 받습니다:

| 파라미터 | 필수 | 설명 |
|-----------|----------|-------------|
| `recipientAddress` | 예 | 입금을 받을 지갑 주소 (유효한 EVM 주소) |
| `toChainId` | 예* | 목적지 체인 ID (예: Base의 경우 8453) |
| `toCurrency` | 예* | 목적지 체인의 토큰 계약 주소 |
| `amount` | 아니오 | wei/최소 단위로 고정된 금액. 제공된 경우 사용자가 금액을 변경할 수 없습니다 |
| `redirect_url` | 아니오 | 완료 후 리디렉션할 HTTPS URL |
| `redirect_label` | 아니오 | 리디렉션 버튼의 사용자 정의 레이블 |
| `partner` | 아니오 | 사용자 정의 구성을 위한 파트너 ID |

\* Hyperliquid 체인에서는 USDC가 기본 토큰으로 설정되어 있어 필요하지 않습니다.

## 예시 URL

**Base USDC로 기본 입금:**
```
https://anyspend.b3.fun/deposit?recipientAddress=0x1234...&toChainId=8453&toCurrency=0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913
```

**고정 금액 입금:**
```
https://anyspend.b3.fun/deposit?recipientAddress=0x1234...&toChainId=8453&toCurrency=0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913&amount=1000000
```

**리디렉션 포함:**
```
https://anyspend.b3.fun/deposit?recipientAddress=0x1234...&toChainId=8453&toCurrency=0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913&redirect_url=https://yourapp.com/success&redirect_label=Return+to+App
```

## 지원되는 지갑

기본적으로 지원됩니다:

- MetaMask
- Coinbase Wallet
- Rainbow
- Rabby
- Trust Wallet

이전에 지갑을 연결한 사용자에게는 자동 연결이 활성화됩니다.

## 특별 체인 지원

### Hyperliquid

Hyperliquid는 특별한 처리를 받습니다:

- USDC는 특별한 34자리 주소 형식(`0x00000000000000000000000000000000`)을 사용합니다.
- `toChainId`가 Hyperliquid로 설정된 경우, `toCurrency`는 지정되지 않은 경우 기본적으로 USDC로 설정됩니다.
- 0 주소는 자동으로 Hyperliquid USDC 형식으로 수정됩니다.

## 파트너 통합

파트너는 사용자 정의 설정으로 구성될 수 있습니다:

```typescript
interface PartnerConfig {
  returnToHomeUrl?: string;      // 기본 리디렉션 URL
  customRecipientLabel?: string; // 수신자에 대한 사용자 정의 레이블
  returnHomeLabel?: string;      // 홈으로 돌아가기 버튼의 사용자 정의 레이블
}
```

파트너로 등록하려면 AnySpend 팀에 연락하여 구성을 추가하십시오.

## 보안

- HTTPS 리디렉션 URL만 허용됩니다 (오픈 리디렉션 공격 방지)
- 금액은 유효한 양의 정수(wei)여야 합니다
- 처리 전에 주소가 유효한지 검증됩니다

## 일반적인 사용 사례

- **텔레그램 봇** -- 공유 가능한 링크를 통해 봇 지갑에 자금을 조달합니다.
- **게임** -- 게임 계정에 토큰을 입금합니다.
- **마켓플레이스** -- 에스크로 주소에 자금을 조달합니다.
- **팁 주기** -- 입금 링크를 통해 사용자가 창작자에게 팁을 줄 수 있습니다.

## 통합 예시

앱에서 입금 링크를 생성합니다:

```typescript
function generateDepositLink({
  recipientAddress,
  chainId,
  tokenAddress,
  amount,
  redirectUrl,
}: {
  recipientAddress: string;
  chainId: number;
  tokenAddress: string;
  amount?: string;
  redirectUrl?: string;
}) {
  const params = new URLSearchParams({
    recipientAddress,
    toChainId: chainId.toString(),
    toCurrency: tokenAddress,
  });

  if (amount) params.set("amount", amount);
  if (redirectUrl) params.set("redirect_url", redirectUrl);

  return `https://anyspend.b3.fun/deposit?${params.toString()}`;
}
```
