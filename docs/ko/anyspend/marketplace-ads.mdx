## Article Submissions

암호화폐 콘텐츠 사이트에 기사 아이디어를 제출하세요. AI가 주제를 검증하고 최적화된 제목과 설명을 생성합니다.

### How It Works

<Steps>
  <Step title="Submit Topic or Tweet">
    트윗 URL, 주제 아이디어 또는 완성된 기사 제목 제공
  </Step>
  <Step title="AI Validation (Free)">
    AI가 콘텐츠를 점수화(0-100)하고 필요한 경우 제안 생성
  </Step>
  <Step title="Get Approval">
    - **점수 ≥80:** 입력이 완벽함, 그대로 사용됨
    - **점수 70-79:** AI가 개선된 제안과 함께 승인됨
    - **점수 &lt;70:** 거부됨 - 다른 주제 시도
  </Step>
  <Step title="Submit & Pay">
    기사 요청을 제출하기 위해 x402 서명 한 번
  </Step>
  <Step title="Publication">
    사이트가 기사를 게시하고 실제 URL을 받음
  </Step>
</Steps>

### Pricing

사이트별로 가격이 다르며, 일반적으로 **$3-50 USDC**입니다. 각 사이트는 트래픽 및 오디언스를 기반으로 자체 요금을 설정합니다.

<Note>
  검증은 무료입니다! 결제 전에 원하는 만큼 아이디어를 검증할 수 있습니다.

### Purchase Ad Slot (x402 Payment)

```bash
POST https://x402.anyspend.com/ads/buy
```

x402 가스 없는 결제를 사용하여 프로그래매틱하게 광고 슬롯을 구매하세요. 이 엔드포인트는 x402 결제 권한을 요구합니다.

**Request Body:**

<CodeGroup>
```json Request
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Alt text",
    "title": "Ad title",
    "description": "Description"
  }
}
```

```json Success Response (200)
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json Error Response (400)
{
  "success": false,
  "error": "Slot already booked during this time"
}
```
</CodeGroup>

**Parameters:**
- `productId` (필수) - `/ads/slots`에서의 광고 슬롯 ID
- `buyerAddress` (필수) - 당신의 지갑 주소
- `startTime` (필수) - ISO 8601 형식의 시작 시간
- `endTime` (필수) - ISO 8601 형식의 종료 시간
- `adContent.imageUrl` (필수) - 광고 이미지의 직접 URL
- `adContent.linkUrl` (필수) - 랜딩 페이지 URL
- `adContent.altText` (필수) - 이미지 대체 텍스트
- `adContent.title` (선택) - 광고 제목
- `adContent.description` (선택) - 광고 설명

**JavaScript/TypeScript 예제:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// 지갑 클라이언트 생성
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// x402 결제로 fetch 래핑
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // 최대 1000 USDC
);

// 광고 슬롯 구매
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: 'My ad',
        title: 'Check out our product!',
        description: 'Limited offer'
      }
    })
  }
);

const result = await response.json();
console.log('Booking ID:', result.data.bookingId);
```

<Note>
  `wrapFetchWithPayment` 사용에 대한 자세한 내용은 [x402 SDK documentation](/anyspend/x402-overview)을 참조하세요.
</Note>

### Browse Sites (Free)

```bash
GET https://x402.anyspend.com/ads/sites
```

페이지네이션을 사용하여 모든 사용 가능한 사이트를 나열합니다. 이 엔드포인트는 무료로 호출할 수 있습니다.

Query parameters:
- `page` - 페이지 번호 (기본값: 1)
- `limit` - 페이지 당 결과 수 (기본값: 20, 최대: 100)
- `search` - 이름/도메인/키워드로 검색

### View Your Campaigns (Free)

```bash
GET https://x402.anyspend.com/ads/bookings/buyer/:address
```

활성, 대기 중 및 완료된 모든 광고 예약을 추적합니다. 이 엔드포인트는 무료로 호출할 수 있습니다.

<CodeGroup>
```bash cURL
curl "https://x402.anyspend.com/ads/bookings/buyer/0xYourAddress"
```

```json Response
{
  "success": true,
  "data": [
    {
      "bookingId": "booking_xyz789",
      "siteId": "abc123",
      "slotType": "banner",
      "startTime": "2024-02-01T10:00:00.000Z",
      "endTime": "2024-02-01T13:00:00.000Z",
      "adContent": {
        "imageUrl": "https://mysite.com/ad.png",
        "linkUrl": "https://mysite.com/landing"
      },
      "totalPrice": "3000000",
      "status": "active",
      "paymentTxHash": "0xabcdef...",
      "createdAt": "2024-01-22T16:30:00.000Z"
    }
  ]
}
```
</CodeGroup>

**상태 값:**
- `active` - 현재 실행 중
- `pending` - 미래에 예정됨
- `completed` - 캠페인 종료됨

<Info>
  API 응답의 가격은 USDC wei(6자리 소수점)입니다. USDC 금액을 얻으려면 1,000,000으로 나누세요.
