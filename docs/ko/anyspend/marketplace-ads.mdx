## 기사 제출

암호화폐 콘텐츠 사이트에 기사 아이디어를 제출하세요. AI가 주제를 검증하고 최적화된 제목 및 설명을 생성합니다.

### 작동 방식

<Steps>
  <Step title="주제 또는 트윗 제출">
    트윗 URL, 주제 아이디어 또는 완성된 기사 제목 제공
  </Step>
  <Step title="AI 검증 (무료)">
    AI가 콘텐츠를 점수화(0-100)하고 필요한 경우 제안사항 생성
  </Step>
  <Step title="승인 받기">
    - **점수 ≥80:** 입력이 완벽함, 그대로 사용
    - **점수 70-79:** AI가 개선된 제안과 함께 승인
    - **점수 &lt;70:** 거절 - 다른 주제 시도

  <Step title="제출 및 결제">
    기사 요청을 제출하기 위한 x402 서명 한 번
  </Step>
  <Step title="게시">
    사이트가 기사를 게시하고 실제 URL을 받음
  </Step>
</Steps>

### 광고 슬롯 구매 (x402 결제)

```bash
POST https://x402.anyspend.com/ads/buy
```

x402 가스 없는 결제를 사용하여 프로그래매틱하게 광고 슬롯을 구매하세요. 이 엔드포인트는 x402 결제 권한이 필요합니다.

**요청 본문:**

<CodeGroup>
```json Request
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Alt text",
    "title": "Ad title",
    "description": "Description"
  }
}
```

```json Success Response (200)
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json Error Response (400)
{
  "success": false,
  "error": "Slot already booked during this time"
}
```
</CodeGroup>

**매개변수:**
- `productId` (필수) - `/ads/slots`에서의 광고 슬롯 ID
- `buyerAddress` (필수) - 지갑 주소
- `startTime` (필수) - ISO 8601 형식의 시작 시간
- `endTime` (필수) - ISO 8601 형식의 종료 시간
- `adContent.imageUrl` (필수) - 광고 이미지의 직접 URL
- `adContent.linkUrl` (필수) - 랜딩 페이지 URL
- `adContent.altText` (필수) - 이미지 대체 텍스트
- `adContent.title` (선택사항) - 광고 제목
- `adContent.description` (선택사항) - 광고 설명

**JavaScript/TypeScript 예시:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// 지갑 클라이언트 생성
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// x402 결제로 fetch 래핑
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // 최대 1000 USDC
);

// 광고 슬롯 구매
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: 'My ad',
        title: 'Check out our product!',
        description: 'Limited offer'
      }
    })
  }
);

const result = await response.json();
console.log('Booking ID:', result.data.bookingId);
```

<Note>
  `wrapFetchWithPayment` 사용에 대한 자세한 내용은 [x402 SDK 문서](/anyspend/x402-overview)를 참조하세요.
</Note>

### 사이트 탐색 (무료)

```bash
GET https://x402.anyspend.com/ads/sites
```

페이지네이션을 포함한 모든 사용 가능한 사이트 목록을 호출합니다. 이 엔드포인트는 무료로 호출할 수 있습니다.

쿼리 매개변수:
- `page` - 페이지 번호 (기본값: 1)
- `limit` - 페이지당 결과 수 (기본값: 20, 최대: 100)
- `search` - 이름/도메인/키워드로 검색

<CodeGroup>
```bash cURL
curl "https://x402.anyspend.com/ads/sites?search=defi&limit=10"
```

```javascript JavaScript
const sites = await fetch('https://x402.anyspend.com/ads/sites?page=1&limit=20')
  .then(r => r.json());
```

```json Response
{
  "success": true,
```
