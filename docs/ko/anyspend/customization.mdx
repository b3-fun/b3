모든 핵심 컴포넌트(`<AnySpend>`, `<AnySpendDeposit>`, `<AnySpendCheckout>`, `<AnySpendCheckoutTrigger>`)는 `slots`, `content`, 및 `theme` props를 받습니다.

## Slots

Slots를 사용하면 전체 UI 섹션을 자체 React 컴포넌트로 교체할 수 있습니다. 각 슬롯은 AnySpend에서 props를 받아 사용자 정의 구현에 사용할 수 있습니다.

```typescript title="AnySpendSlots Interface" icon="code"
interface AnySpendSlots {
  /** 메인 액션/결제 버튼 교체 */
  actionButton?: (props: ActionButtonSlotProps) => ReactNode;
  /** 지갑 연결 버튼 교체 */
  connectWalletButton?: (props: ConnectWalletButtonSlotProps) => ReactNode;
  /** 헤더 교체 */
  header?: (props: { mode: "page" | "modal" }) => ReactNode;
  /** 푸터 교체 */
  footer?: ReactNode;
  /** 성공 화면 교체 */
  successScreen?: (props: SuccessScreenSlotProps) => ReactNode;
  /** 오류 화면 교체 */
  errorScreen?: (props: ErrorScreenSlotProps) => ReactNode;
  /** 체크아웃 양식 패널 교체 (체크아웃 전용) */
  checkoutForm?: (props: CheckoutFormComponentProps) => ReactNode;
  /** 배송 방법 선택기 교체 (체크아웃 전용) */
  shippingSelector?: (props: ShippingSelectorSlotProps) => ReactNode;
  /** 할인 코드 입력 교체 (체크아웃 전용) */
  discountInput?: (props: DiscountInputSlotProps) => ReactNode;
}
```

<Tip>
  `checkoutForm`, `shippingSelector`, 및 `discountInput` 슬롯은 `<AnySpendCheckout>` 및 `<AnySpendCheckoutTrigger>`에서만 사용됩니다. 양식/배송/할인 기능에 대한 자세한 내용은 [체크아웃 가이드](/anyspend/checkout)를 참조하세요.
</Tip>

## Content

다양한 거래 상태에서 표시되는 텍스트와 메시지를 재정의합니다. 각 필드는 `string` 또는 `ReactNode`를 받습니다.

```typescript title="AnySpendContent Interface" icon="code"
interface AnySpendContent {
  // 성공 상태
  successTitle?: string | ReactNode;
  successDescription?: string | ReactNode;

  // 실패 상태
  failureTitle?: string | ReactNode;
  failureDescription?: string | ReactNode;

  // 만료 상태
  expiredTitle?: string | ReactNode;
  expiredDescription?: string | ReactNode;

  // 환불 상태
  refundedTitle?: string | ReactNode;
  refundedDescription?: string | ReactNode;

  // 처리 중 상태
  processingTitle?: string | ReactNode;
  processingDescription?: string | ReactNode;

  // 버튼 레이블
  returnButtonLabel?: string;
  retryButtonLabel?: string;
}
```

## Theme

브랜드 색상으로 AnySpend 컴포넌트의 시각적 외관을 구성합니다.

```typescript title="AnySpendTheme Interface" icon="code"
interface AnySpendTheme {
  /** 기본 브랜드 색상 (hex). 주요 강조 색상으로 적용됩니다. */
  brandColor?: string;
  /** 개별 색상 재정의 (hex 값) */
  colors?: Partial<{
    primary: string;         // 주요 텍스트 색상
    secondary: string;       // 보조 텍스트 색상
    tertiary: string;        // 미묘한 텍스트 색상
    surfacePrimary: string;  // 기본 배경 색상
    surfaceSecondary: string; // 보조 배경 (카드, 패널)
    brand: string;           // 강조/브랜드 색상 (버튼, 링크)
    borderPrimary: string;   // 기본 테두리 색상
    borderSecondary: string; // 보조 테두리 색상
  }>;
}
```

<Note>
  `brandColor` prop은 `brand` 색상을 설정하는 단축키입니다. `brandColor`와 `colors.brand` 둘 다 제공되는 경우, `colors.brand`가 우선합니다.
</Note>

## CSS 클래스 재정의

AnySpend 컴포넌트 내 특정 요소에 사용자 정의 CSS 클래스를 적용합니다. 각 컴포넌트는 자체 클래스 인터페이스를 가집니다.

### AnySpendClasses

`<AnySpend>`에서 사용됩니다:

```typescript title="AnySpendClasses" icon="code"
interface AnySpendClasses {
  container?: string;
  header?: string;
  tabSection?: string;
  cryptoPaySection?: string;
  cryptoReceiveSection?: string;
  panelOnramp?: string;
  paymentButton?: string;
  navigationBar?: string;
}
```
