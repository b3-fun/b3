Daily Slots는 매일 무료 보상을 위해 운을 시험해 볼 수 있는 빠르고 쉬운 방법입니다. 로그인하고 레버를 당긴 다음, 릴이 결정하게 하세요. 이긴 토큰은 클레임할 필요 없이 직접 지갑으로 보내집니다.

## Daily Slots 플레이 방법

<Steps>
<Step title="스테이킹 플랫폼으로 이동">
  [stake.b3.fun](http://stake.b3.fun)으로 가서 선호하는 로그인 방법을 사용하여 로그인하세요.
</Step>

<Step title="일일 슬롯 접근">
  메인 페이지에서 **Free Slots!** 아이콘을 클릭하세요.
  
  <Frame>
    <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/685306b72e0550277aa9250e/images/688105a2a309495486d153c7/file-YRmsPT7ITx.png" alt="스테이킹 대시보드의 Free Slots 아이콘" />
  </Frame>
</Step>

<Step title="레버 당기기">
  일일 슬롯 페이지에서 **Pull the lever.**를 클릭하세요.
  
  <Frame>
    <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/685306b72e0550277aa9250e/images/68810603b28ecf16639c8d18/file-0XCnrc6tZH.png" alt="슬롯 인터페이스의 레버 당기기 버튼" />
  </Frame>
</Step>

<Step title="트랜잭션 확인">
  연결된 지갑에서 트랜잭션을 확인하여 스핀을 완료하세요.
  
  <Check>
    보상은 자동으로 지갑으로 보내집니다!
  </Check>
</Step>
</Steps>

## 게임 규칙

- **지갑당 하루에 한 번 플레이**
- **클레임 필요 없음** - 보상은 직접 지갑으로 보내집니다
- **적어도 1,000 B3를 스테이킹해야** 자격이 있음
- **완전 무료** - 추가 비용 없음

<Info>
Daily Slots는 빠르고 재미있게 설계되었습니다 - B3 보너스 보상을 받을 기회를 위해 하루에 한 번만 클릭하세요!
</Info>
