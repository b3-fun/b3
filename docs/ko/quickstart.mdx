### 1단계: B3 SDK 설치
<Steps>
  <Step title="SDK 패키지 설치">
    선호하는 패키지 관리자를 사용하여 B3 SDK를 설치하세요:

    <CodeGroup>
    ```bash npm
    npm install @b3dotfun/sdk
    ```
    ```bash pnpm
    pnpm add @b3dotfun/sdk
    ```
    ```bash yarn
    yarn add @b3dotfun/sdk
    ```
    </CodeGroup>

    <Tip>B3 SDK는 TypeScript 정의를 기본으로 포함하고 있습니다!</Tip>
  </Step>

  <Step title="CSS 스타일 가져오기">
    애플리케이션에 기본 스타일을 가져옵니다:

    ```typescript
    import "@b3dotfun/sdk/index.css";
    ```
  </Step>
</Steps>

### 2단계: B3Provider 추가
애플리케이션의 루트 주변에 B3Provider를 감싸세요:

```tsx
import { B3Provider, SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

function App() {
  return (
    <B3Provider environment="production" partnerId="your-awesome-partner-id">
      {children}
    </B3Provider>
  );
}
```


### 3단계: B3 SDK 사용
<AccordionGroup>
 <Accordion icon="swap" title="즉시 계정">
    B3를 사용한 글로벌 계정 인증 활성화:

    ```tsx
    import { SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

    function App() {
      return (
        <B3Provider environment="production" partnerId="your-awesome-partner-id">
          <SignInWithB3
            provider={{ strategy: "google" }}
            partnerId="your-partner-id"
            onLoginSuccess={(globalAccount) => {
              console.log("Authenticated:", globalAccount);
            }}
          />
        </B3Provider>
      );
    }
    ```
  </Accordion>
  <Accordion icon="swap" title="유동성 접근">
    AnySpend를 사용한 크로스체인 스왑 및 사용자 정의 상호작용 활성화:

    ```tsx
    import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

    function SwapPage() {
      return <AnySpend mode="page" />;
    }
    ```
  </Accordion>
</AccordionGroup>
