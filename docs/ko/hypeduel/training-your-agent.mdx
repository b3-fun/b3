## 시작하기

### 필수 조건

에이전트를 훈련하기 전에 다음을 준비하세요:

- **Unity 2022.3 LTS** 또는 그 이상
- **Python 3.8+** 및 pip 설치됨
- **ML-Agents Toolkit** from Unity
- **PyTorch** 또는 **TensorFlow** for training

### 설치 단계

<Steps>
  <Step title="Unity ML-Agents 설치">
    ```bash
    pip install mlagents
    ```
  </Step>
  <Step title="HypeDuel 훈련 템플릿 클론">
    ```bash
    git clone https://github.com/hypeduel/agent-training-template
    cd agent-training-template
    ```
  </Step>
  <Step title="의존성 설치">
    ```bash
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Unity에서 열기">
    Unity Hub에서 프로젝트를 열고 훈련 장면을 로드하세요
  </Step>
</Steps>

## 훈련 구성

### 에이전트 행동 파라미터

`config.yaml` 파일에서 에이전트의 학습 파라미터를 구성하세요:

```yaml
behaviors:
  HypeDuelAgent:
    trainer_type: ppo
    hyperparameters:
      batch_size: 1024
      buffer_size: 10240
      learning_rate: 3.0e-4
      beta: 5.0e-4
      epsilon: 0.2
      lambd: 0.99
      num_epoch: 3
      learning_rate_schedule: linear
    network_settings:
      normalize: false
      hidden_units: 128
      num_layers: 2
    reward_signals:
      extrinsic:
        gamma: 0.99
        strength: 1.0
    max_steps: 500000
    time_horizon: 64
    summary_freq: 10000
```

### 관찰 공간

에이전트는 다음을 포함한 관찰을 받게 됩니다:

- **아레나 상태**: 현재 위치, 체력 및 파워업
- **상대 행동**: 이동 패턴 및 최근 행동
- **환경 데이터**: 아레나 경계, 장애물 및 위험
- **게임 상태**: 남은 시간, 점수 차이 및 파워업 가능성

### 행동 공간

에이전트는 다음과 같은 행동을 수행할 수 있습니다:

- **이동**: 앞으로, 뒤로, 왼쪽, 오른쪽
- **전투**: 공격, 방어, 특수 능력
- **전략**: 위치 선정, 파워업 수집, 방어적 기동

## 훈련 과정

### 로컬 훈련

에이전트를 로컬에서 훈련시키세요:

```bash

```

### 아레나 제출

에이전트 훈련이 완료되면:

1. **성능 테스트**: 벤치마크 상대에 대해 검증
2. **모델 최적화**: 효율적인 추론 속도 확보
3. **HypeDuel에 제출**: `.onnx` 모델 파일 업로드
4. **아레나 등록**: 특정 아레나에 에이전트 등록

## 모범 사례

### 훈련 팁

- **단순하게 시작**: 복잡한 전략 전에 기본적인 행동부터 시작하세요
- **보상 엔지니어링**: 원하는 행동을 장려하는 보상 설계
- **정기적 평가**: 다양한 상대에 대해 자주 테스트하세요
- **점진적 복잡성**: 고급 메커닉을 점차적으로 도입하세요

### 성능 최적화

- **관찰 정규화**: 안정적인 훈련을 위한 입력 스케일링
- **행동 공간 설계**: 표현력과 훈련 효율성 사이의 균형
- **메모리 관리**: 훈련 및 추론 모두에 대한 최적화
- **모델 압축**: 빠른 아레나 배포를 위한 모델 크기 축소

## 문제 해결

### 일반적인 문제

<Accordion title="훈련이 수렴하지 않음">
  학습률, 보상 스케일링 및 관찰 정규화를 확인하세요. 복잡성을 줄이거나 하이퍼파라미터를 조정하는 것을 고려하세요.
</Accordion>

<Accordion title="에이전트 성능 정체">
  커리큘럼 학습을 구현하거나 환경 다양성을 증가시키거나, 더 강한 상대에 대한 자체 플레이 훈련을 시도하세요.
</Accordion>

<Accordion title="추론 속도가 너무 느림">
  모델 아키텍처를 최적화하거나 ONNX 런타임을 사용하거나 반복 결정에 대한 행동 캐싱을 구현하세요.
</Accordion>
