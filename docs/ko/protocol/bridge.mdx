지갑을 연결하고 주요 EVM 네트워크에서 B3으로 입출금하세요.

<Steps>
<Step title="Basement Bridge로 이동">
  브릿지 인터페이스에 접근하려면 [basement.fun/bridge](https://basement.fun/bridge)를 방문하세요.
</Step>

<Step title="지갑 연결">
  MetaMask, Coinbase Wallet 또는 지원되는 다른 지갑을 사용하여 지갑을 연결하세요.
</Step>

<Step title="네트워크 및 자금 선택">
  출처 네트워크와 B3로 브릿지할 자금의 양을 선택하세요.
</Step>

<Step title="거래 확인">
  거래를 확인하면 1분 이내에 브릿지된 자금을 받게 됩니다.
  
  <Check>
  귀하의 자금은 60초 이내에 B3 지갑에 나타납니다.
  </Check>
</Step>
</Steps>

<Steps>

  </Check>

## Native Bridge

<div style={{ marginBottom: '16px' }}>
  <span style={{ backgroundColor: '#3b82f6', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Base to B3</span>
  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>7일 출금</span>
  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>수수료 없음</span>
</div>

공식 OP Stack canonical bridge를 사용하여 Base에서 B3로 쉽게 브릿지하세요!

<Warning>
B3에서 Base로 다시 출금하는 경우 보안을 위해 7일간의 챌린지 기간이 있습니다. 즉시 출금을 위해서는 타사 브릿지를 사용하세요.
</Warning>

<Steps>
<Step title="브릿지 방문">
  공식 브릿지에 접근하려면 [bridge.b3.fun](https://bridge.b3.fun)을 방문하세요.
</Step>

<Step title="지갑 연결">
  브릿지 인터페이스에 지갑을 연결하세요.
</Step>

<Step title="방향 선택">
  다음 중 하나를 선택하세요:
  - **입금**: Base → B3로 자금을 가져옵니다
  - **출금**: B3에서 Base로 브릿지합니다
</Step>

<Step title="금액 입력">
  브릿지할 ETH의 양을 입력하세요.
</Step>

<Step title="거래 완료">
  "입금"을 클릭하고 지갑에서 거래를 확인하세요.
  
  <Tip>
  B3로의 입금은 즉시 이루어지지만, 출금은 챌린지 기간 동안 7일이 걸립니다.
  </Tip>
</Step>
</Steps>

</Card>

## Native Transfer (Bridge Contract)

  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>7일 출금</span>

다음 브릿지 주소로 **Base Mainnet**에서 **Base ETH**를 전송하세요:

<Warning>
이 방법은 고급 사용자만을 위한 것입니다. 자금을 잃지 않도록 Base Mainnet에서 전송하는지 확인하세요.
</Warning>

```text Bridge Address
0x769547a723783FCA36BAaf1ECcf9dfdbF6d09F38
```

<Note>
Base에서 이 주소로 ETH를 보내면 자동으로 B3의 지갑에 나타납니다. 이는 직접 계약 상호작용 방법입니다.
</Note>

<Accordion title="브릿지 계약 사용 방법">
1. **지갑에서 Base Mainnet에 있어야 합니다**
2. **위의 브릿지 계약 주소로 ETH를 보냅니다**
3. **확인을 기다립니다** - 귀하의 자금은 B3에서 자동으로 나타납니다
4. **B3 지갑 확인** - 브릿지된 ETH를 보려면 B3 네트워크로 전환하세요

<Warning>
전송하기 전에 Base Mainnet에 있는지 확인하세요. 다른 네트워크에서 전송하면 자금이 손실될 수 있습니다.
</Warning>
</Accordion>
