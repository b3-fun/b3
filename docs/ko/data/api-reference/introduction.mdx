## 데이터 소스 & 인텔리전스

<Steps>
  <Step title="Blockscout 통합">
    모든 B3 체인의 Blockscout 탐색기에서 색인화된 블록체인 데이터에 직접 액세스하여 거래 기록, 계약 상호작용 및 블록 데이터를 제공합니다.
  </Step>
  <Step title="Thirdweb 데이터 레이어">
    NFT 메타데이터, 계약 ABIs, 스마트 계약 분석을 위한 Thirdweb의 강력한 데이터 인프라로 강화됩니다.
  </Step>
  <Step title="지능적 라우팅">
    우리의 API는 요청을 최적의 데이터 소스 및 체인으로 자동 라우팅하여 빠른 응답 시간과 데이터 정확성을 보장합니다.
  </Step>
  <Step title="통합된 생태계 뷰">
    모든 B3 생태계 체인에서 병합된 데이터는 크로스 체인 활동과 자산의 포괄적인 뷰를 제공합니다.
  </Step>
</Steps>
