## 字体

在 CSS 样式表中导入：

```css
@import url("https://cdn.b3.fun/fonts.css");
```

通过 `<link>` 标签导入：

```html
<link href="https://cdn.b3.fun/fonts.css" rel="stylesheet">
```

现在您可以使用这些字体：

```css
body { font-family: "Neue Montreal Medium", sans-serif; }
```

#### 可用字体：

- **Neue Montreal Bold**  
- **Neue Montreal Regular**  
- **Neue Montreal Book**  
- **Neue Montreal Medium**  
- **Neue Montreal Semibold**  
- **Calibre Bold**  
- **Calibre Heavy**  
- **Pack**
