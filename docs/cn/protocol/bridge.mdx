## 原生桥接

<div style={{ marginBottom: '16px' }}>
  <span style={{ backgroundColor: '#3b82f6', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Base 到 B3</span>
  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>7天提款</span>
  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>无费用</span>
</div>

使用官方 OP Stack 规范桥轻松从 Base 到 B3！

<Warning>
从 B3 返回 Base 的提款有 7 天的挑战期以确保安全。使用第三方桥接以实现即时提款。
</Warning>

<Steps>
<Step title="访问桥接">
  前往 [bridge.b3.fun](https://bridge.b3.fun) 访问官方桥接。
</Step>

<Step title="连接您的钱包">
  将您的钱包连接到桥接界面。
</Step>

<Step title="选择方向">
  选择您想要的操作：
  - **存款**：从 Base → B3
  - **提款**：从 B3 → Base
</Step>

<Step title="输入金额">
  输入您想要桥接的 ETH 金额。
</Step>

<Step title="完成交易">
  点击“存款”并在您的钱包中确认交易。
  
  <Tip>
  到 B3 的存款是即时的，但提款需要 7 天的挑战期。
  </Tip>
</Step>
</Steps>

通过桥接合约发送 Base ETH。将 **Base ETH** 发送到以下地址 **Base Mainnet** 上的桥接地址：

<Warning>
此方法仅适用于高级用户。确保您是从 Base Mainnet 发送，以避免资金损失。
</Warning>

```text Bridge Address
0x769547a723783FCA36BAaf1ECcf9dfdbF6d09F38
```

<Note>
当您从 Base 向此地址发送 ETH 时，它将自动出现在您的 B3 钱包中。这是一种直接合约交互方法。
</Note>

<Accordion title="如何使用桥接合约">
1. **确保您的钱包处于 Base Mainnet** 
2. **向上面的桥接合约地址发送 ETH** 
3. **等待确认** - 您的资金将自动出现在 B3 上
4. **检查您的 B3 钱包** - 切换到 B3 网络查看您的桥接 ETH

<Warning>
发送前请再次检查您是否在 Base Mainnet 上。从其他网络发送可能会导致资金损失。
</Warning>
</Accordion>
