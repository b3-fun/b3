### 创建您的 B3 全球账户

<Steps>
  <Step title="访问 Explorer">
    访问 https://explorer.b3.fun/。
  </Step>
  <Step title="打开侧边栏">
    在页面的左侧，悬停以展开侧边栏。选择任意选项（如 **交易** 或 **浏览应用**）以显示登录选项。

    ![Screenshot2025 09 24at6 23 08PM Pn](/docs/images/Screenshot2025-09-24at6.23.08PM.png)
  </Step>
  <Step title="连接您的账户">
    在屏幕的右上角，点击 **连接**。

    ![Screenshot2025 09 24at6 26 24PM Pn](/docs/images/Screenshot2025-09-24at6.26.24PM.png)
  </Step>
  <Step title="选择登录方式">
    - **钱包 (MetaMask, WalletConnect 等)** - 在您的钱包中批准连接请求。
    - **社交 / 邮箱 / 电话** - 跟随适合您登录方式的验证过程（例如，电子邮件确认、短信代码或社交登录授权）。

    <img
      src="/docs/images/Screenshot2025-09-24at6.28.03PM.png"
      alt="Screenshot2025 09 24at6 28 03PM Pn"
      title="Screenshot2025 09 24at6 28 03PM Pn"
      className="mx-auto"
      style={{ width:"61%" }}
    />
  </Step>
</Steps>

一旦验证通过，您的 B3 全球账户将被创建，并且您将被登录。

### 常见问题解答

<Accordion title="我需要一个钱包来创建 B3 全球账户吗？">
  不。您可以使用钱包、邮箱、电话或社交登录来注册。如果您选择不使用钱包登录，一个嵌入式钱包将自动为您在后台创建。这确保了即使您没有带自己的钱包，您的账户仍然连接有一个钱包。
</Accordion>
