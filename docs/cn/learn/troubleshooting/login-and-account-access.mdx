## 常见问题解答

<AccordionGroup>
<Accordion title="为什么我无法登录我的账户？">
请确保您使用了正确的登录方式，并且您的钱包或社交账户已正确连接。
</Accordion>

<Accordion title="我没有收到验证邮件。我该怎么办？">
检查垃圾邮件或尝试重新发送。如果仍然没有收到，请尝试使用不同的登录方式，如钱包或Discord，或通过我们的支持门户联系我们以获得额外的帮助。
</Accordion>

<Accordion title="支持哪些登录方式？">
您可以使用电子邮件、Discord、Telegram、钱包或如Google和Apple这样的社交账户登录。
</Accordion>

<Accordion title="我可以将多个账户链接到一个个人资料中吗？">
是的。您可以在[**basement.fun**](https://basement.fun/)上的个人资料设置中连接您的钱包、Discord和其他账户。
</Accordion>
</AccordionGroup>

<Info>
如果您继续遇到登录问题，请确保您使用的是最初创建账户时使用的相同登录方式。
</Info>
