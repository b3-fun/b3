## 基本设置

### 1. 提供者设置

用 `B3Provider` 和 `AnyspendProvider` 包裹您的应用，以启用 AnySpend 功能：

```tsx App.tsx icon=react
import { AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";
import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production">
      <AnyspendProvider>{/* 您的应用组件 */}</AnyspendProvider>
    </B3Provider>
  );
}

export default App;
```

### 3. TypeScript 配置（可选但推荐）

<Note>
  如果您使用 TypeScript，请确保您的 `tsconfig.json` 包含以下设置，以获得最佳兼容性：
</Note>

```json tsconfig.json icon=settings
{
  "compilerOptions": {
    "moduleResolution": "node",
    "allowSyntheticDefaultImports": true,
    "esModuleInterop": true
  }
}
```

## 故障排除

<AccordionGroup>
  <Accordion title="导入错误">
    确保您已正确安装 SDK 并导入了 CSS 文件。SDK 需要 React 18+，并且可能与旧版本存在兼容性问题。
  </Accordion>
  <Accordion title="未找到提供者">
    确保您在组件树的高层，通常在您的主应用组件中，放置了 `B3Provider` 和 `AnyspendProvider`。
  </Accordion>
  <Accordion title="TypeScript 错误">
