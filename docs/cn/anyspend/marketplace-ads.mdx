### 购买广告位（x402 支付）

```bash
POST https://x402.anyspend.com/ads/buy
```

使用 x402 无气费支付以编程方式购买广告位。此端点需要 x402 支付授权。

**请求正文：**

<CodeGroup>
```json 请求
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Alt 文本",
    "title": "广告标题",
    "description": "描述"
  }
}
```

```json 成功响应（200）
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json 错误响应（400）
{
  "success": false,
  "error": "该时间段已被预订"
}
```
</CodeGroup>

**参数：**
- `productId`（必填）- 广告位 ID，来自 `/ads/slots`
- `buyerAddress`（必填）- 您的钱包地址
- `startTime`（必填）- 开始时间，ISO 8601 格式
- `endTime`（必填）- 结束时间，ISO 8601 格式
- `adContent.imageUrl`（必填）- 广告图片的直接 URL
- `adContent.linkUrl`（必填）- 落地页 URL
- `adContent.altText`（必填）- 图片替代文本
- `adContent.title`（可选）- 广告标题
- `adContent.description`（可选）- 广告描述

**JavaScript/TypeScript 示例：**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// 创建钱包客户端
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// 用 x402 支付包装 fetch
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // 最大 1000 USDC
);

// 购买广告位
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: '我的广告',
        title: '看看我们的产品！',
        description: '限时优惠'
      }
    })
  }
);

const result = await response.json();
console.log('预订 ID:', result.data.bookingId);
```

<Note>
  有关使用 `wrapFetchWithPayment` 的更多详情，请参阅 [x402 SDK 文档](/anyspend/x402-overview)。
</Note>
