## AI 验证规则

AI 验证端点对您的文章想法进行评分，并将其按原样批准、建议改进或拒绝。

<Tabs>
  <Tab title="✅ 分数 ≥80">
    **完美！** 您的输入按原样获得批准。
    
    您提供的标题或主题非常优秀，符合站点的重点。无需更改 - 我们将按照您编写的内容提交。
    
    **示例：** "如何在您的 DApp 中使用 Base Pay 为创作者"
  </Tab>
  <Tab title="⚠️ 分数 70-79">
    **带建议的好！** AI 提供了改进的版本。
    
    您的内容相关但可以优化。AI 生成更好的标题和描述，同时保留您的核心想法。
    
    **示例输入：** "为创作者提供 base pay"
    
    **AI 输出：** "整合 Base Pay：内容创作者的完整指南"
  </Tab>
  <Tab title="❌ 分数 <70">
    **被拒绝。** 内容不符合站点的重点。
    
    您的主题与该站点的主题不相关。尝试不同的主题或选择更符合您内容的不同站点。
    
    **示例：** 向加密新闻站点提交烹饪食谱
  </Tab>
</Tabs>

### 提交文章请求（x402 支付）

```bash
POST https://x402.anyspend.com/articles/buy
```

使用 x402 支付提交文章请求。您必须先验证您的内容并获得批准。

**请求体：**

<CodeGroup>
```json 请求
{
  "buyerAddress": "0x742d35Cc6...",
  "siteDomain": "baseradar.com",
  "title": "整合 Base Pay：内容创作者的完整指南",
  "description": "通过逐步示例和最佳实践学习如何在您的创作者平台中实施 Base Pay。"
}
```

```json 成功响应（200）
{
  "success": true,
  "data": {
    "requestId": "req_789...",
    "productSpecificId": "req_789...",
    "totalPrice": "50000000",
    "status": "pending"
  }
}
```

```json 错误响应（400）
{
  "success": false,
  "error": "提交前需要进行内容验证"
}
```
</CodeGroup>

**参数：**
- `buyerAddress`（必需）- 您的钱包地址
- `siteDomain`（必需）- 目标站点域名
- `title`（必需）- 文章标题（来自验证或您自己的，如果分数 ≥90）
- `description`（必需）- 文章描述（来自验证或您自己的）

**JavaScript/TypeScript 示例：**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// 步骤 1：验证您的内容（免费）
const validateResponse = await fetch('https://x402.anyspend.com/articles/validate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    siteDomain: 'baseradar.com',
    userInput: '如何为创作者使用 Base Pay'
  })
});

const validation = await validateResponse.json();

if (!validation.data.approved) {
  console.error('内容被拒绝：', validation.data.reasoning);
  process.exit(1);
}

console.log('✅ 已批准！分数：', validation.data.score);
console.log('标题：', validation.data.suggestedTitle);

// 步骤 2：使用支付提交（x402）
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // 最多 1000 USDC
);

const response = await fetchWithPayment(
  'https://x402.anyspend.com/articles/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      buyerAddress: '0x742d35Cc6...',
      siteDomain: 'baseradar.com',
      title: validation.data.suggestedTitle,
      description: validation.data.suggestedDescription
    })
  }
);

const result = await response.json();
console.log('🎉 文章已提交！请求 ID：', result.data.requestId);
```

<Note>
  有关使用 `wrapFetchWithPayment` 的更多详情，请参阅 [x402 SDK 文档](/anyspend/x402-overview)。
</Note>
