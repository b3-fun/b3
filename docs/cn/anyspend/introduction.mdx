## AnySpend 是什么？

AnySpend 是 B3 提供的全面支付基础设施，它能够在多个区块链之间无缝进行加密货币交易。无论您是在构建 DeFi 应用、NFT 市场还是游戏平台，AnySpend 都提供了您需要接受任何代币支付的组件和 API。

[在您的浏览器中尝试 AnySpend](https://anyspend.com/swap)

## 快速开始

只需几个步骤即可开始使用 AnySpend：

```tsx
import { AnySpend, AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";

import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production">
      <AnyspendProvider>
        <AnySpend />
      </AnyspendProvider>
    </B3Provider>
  );
}

export default App
```
