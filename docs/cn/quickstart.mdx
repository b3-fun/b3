### 步骤 1：安装 B3 SDK
<Steps>
  <Step title="安装 SDK 包">
    选择您偏好的包管理器来安装 B3 SDK：

    <CodeGroup>
    ```bash npm
    npm install @b3dotfun/sdk
    ```
    ```bash pnpm
    pnpm add @b3dotfun/sdk
    ```
    ```bash yarn
    yarn add @b3dotfun/sdk
    ```
    </CodeGroup>

    <Tip>B3 SDK 自带 TypeScript 定义！</Tip>
  </Step>

  <Step title="导入 CSS 样式">
    在您的应用程序中导入默认样式：

    ```typescript
    import "@b3dotfun/sdk/index.css";
    ```
  </Step>
</Steps>

### 步骤 2：添加 B3Provider
在您应用程序的根部署 B3Provider：

```tsx
import { B3Provider, SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

function App() {
  return (
    <B3Provider environment="production">
      {children}
    </B3Provider>
  );
}
```


### 步骤 3：使用 B3 SDK
<AccordionGroup>
 <Accordion icon="swap" title="即时账户">
    使用 B3 启用全球账户认证：

    ```tsx
    import { SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

    function App() {
      return (
        <B3Provider environment="production">
          <SignInWithB3
            provider={{ strategy: "google" }}
            partnerId="your-partner-id"
            onLoginSuccess={(globalAccount) => {
              console.log("Authenticated:", globalAccount);
            }}
          />
        </B3Provider>
      );
    }
    ```
  </Accordion>
  <Accordion icon="swap" title="访问流动性">
    启用跨链交换 & 自定义交互 AnySpend：

    ```tsx
    import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

    function SwapPage() {
      return <AnySpend mode="page" />;
    }
    ```
  </Accordion>
</AccordionGroup>
