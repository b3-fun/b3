## Visão Geral

AnySpend x402 suporta pagamentos em várias redes blockchain, incluindo cadeias compatíveis com EVM e Solana (SVM). Esta página fornece a referência completa das redes suportadas, tokens e seus endereços de contrato.

AnySpend x402 suporta as seguintes cadeias principais EVM:

| Rede | Identificador | Chain ID | Tipo |

| Abstract | `abstract` | 2741 | L2 |
| Arbitrum One | `arbitrum` | 42161 | L2 - Optimistic Rollup |

| B3 | `b3` | 8333 | L2 |

## Compatibilidade de Tokens

<Note>
Para usar um token com X402, ele deve implementar **EIP-3009** (`transferWithAuthorization`) ou **EIP-2612** (`permit`) para assinaturas sem gás.
</Note>

Em vez de manter uma lista estática de tokens suportados, construímos ferramentas para ajudá-lo a verificar a compatibilidade de qualquer token em tempo real.

## Verificador de compatibilidade de tokens X402

<Note>
**Nova Ferramenta Disponível**: Verifique se qualquer token ERC-20 é compatível com pagamentos X402 antes da integração.
</Note>

Para um token ser compatível com X402, ele deve implementar:
- **EIP-3009** (`transferWithAuthorization`) - Usado por USDC
- **EIP-2612** (`permit`) - Usado por USDT (em algumas cadeias) e muitos outros tokens ERC-20 modernos

### Usando o verificador público

Verifique qualquer token ERC-20 em cadeias suportadas pelo AnySpend:

**Interface web**: [anyspend.com/x402-tokens](https://anyspend.com/x402-tokens)

Basta inserir um endereço de token e selecionar a rede para verificar instantaneamente a compatibilidade X402.

**API Direta**: Você também pode consultar a API diretamente:

```bash
curl https://tokens.anyspend.com/metadata/base/0xb3b32f9f8827d4634fe7d973fa1034ec9fddb3b3
```

Isso retorna os metadados do token e informações de compatibilidade em formato JSON.

### Usando o pacote npm

Para desenvolvedores que integram verificações de compatibilidade de tokens em suas aplicações:

**pacote npm**: [@b3dotfun/anyspend-x402-token-compat](https://www.npmjs.com/package/@b3dotfun/anyspend-x402-token-compat)

```bash
npm install @b3dotfun/anyspend-x402-token-compat
```

**Uso:**
```typescript
import { TokenCompatClient } from '@b3dotfun/anyspend-x402-token-compat';

const client = new TokenCompatClient();

// Obter metadados do token e compatibilidade
const metadata = await client.getTokenMetadata('base', '0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913');
console.log(metadata.supportsEip2612); // true - suporta permit
console.log(metadata.supportsEip3009); // true - suporta transferWithAuth

// Verificação rápida de compatibilidade
const hasPermit = await client.supportsEip2612('base', tokenAddress);
```


### Como funciona

**Rotas Cross-Chain Suportadas:**
AnySpend suporta roteamento cross-chain entre quaisquer das redes EVM suportadas, incluindo:
- Ethereum ↔ Base, Arbitrum, Optimism, Polygon, Avalanche
- Base ↔ Arbitrum, Optimism, Polygon
- L1 ↔ L2 bridging em todas as cadeias suportadas
- Cross-VM: Solana ↔ cadeias EVM (experimental)

<Info>
Pagamentos cross-chain levam mais tempo (2-5 minutos) devido ao tempo de bridging e incorrem em taxas adicionais (~0,5% total). Pagamentos na mesma cadeia são recomendados para a melhor experiência do usuário.
</Info>


<Note>
Construído na infraestrutura global da Cloudflare com cache inteligente para resultados instantâneos.
</Note>

O verificador combina verificação on-chain em tempo real com um banco de dados crescente de tokens verificados:

- Tempos de resposta em nível de milissegundos através de cache inteligente
- Banco de dados cresce automaticamente à medida que mais tokens são verificados
- Verificação on-chain para tokens não armazenados em cache
- Executa em Cloudflare Workers para acesso de baixa latência em todo o mundo

### Integração no x402

O verificador de compatibilidade de tokens já está integrado no SDK AnySpend X402:

- Valida a compatibilidade do token antecipadamente antes do pagamento
- Retorna erros claros e acionáveis para tokens não suportados
- Previne transações falhas devido a tokens incompatíveis

**Experimente ao vivo**: [https://x402-demo.anyspend.com/](https://x402-demo.anyspend.com/)

<Warning>
**Importante**: Sempre verifique a compatibilidade do token antes de integrar um novo token ao seu fluxo de pagamento. Tokens incompatíveis resultarão em falhas nas assinaturas de pagamento.
</Warning>
