## Slots

Slots permitem substituir seções inteiras da UI por seus próprios componentes React. Cada slot recebe props do AnySpend que você pode usar em sua implementação personalizada.

```typescript title="Interface AnySpendSlots" icon="code"
interface AnySpendSlots {
  /** Substitui o botão principal de ação/pagamento */
  actionButton?: (props: ActionButtonSlotProps) => ReactNode;
  /** Substitui o botão de conectar carteira */
  connectWalletButton?: (props: ConnectWalletButtonSlotProps) => ReactNode;
  /** Substitui o cabeçalho */
  header?: (props: { mode: "page" | "modal" }) => ReactNode;
  /** Substitui o rodapé */
  footer?: ReactNode;
  /** Substitui a tela de sucesso */
  successScreen?: (props: SuccessScreenSlotProps) => ReactNode;
  /** Substitui a tela de erro */
  errorScreen?: (props: ErrorScreenSlotProps) => ReactNode;
  /** Substitui o painel do formulário de checkout (somente checkout) */
  checkoutForm?: (props: CheckoutFormComponentProps) => ReactNode;
  /** Substitui o seletor de método de envio (somente checkout) */
  shippingSelector?: (props: ShippingSelectorSlotProps) => ReactNode;
  /** Substitui a entrada do código de desconto (somente checkout) */
  discountInput?: (props: DiscountInputSlotProps) => ReactNode;
}
```

<Tip>
  Os slots `checkoutForm`, `shippingSelector` e `discountInput` são usados apenas em `<AnySpendCheckout>` e `<AnySpendCheckoutTrigger>`. Veja o [Guia de Checkout](/anyspend/checkout) para detalhes sobre recursos de formulário/envio/desconto.
</Tip>

## Tema

Configure a aparência visual dos componentes AnySpend com cores da marca.

```typescript title="Interface AnySpendTheme" icon="code"
interface AnySpendTheme {
  /** Cor primária da marca (hex). Aplicada como a cor principal de destaque. */
  brandColor?: string;
  /** Sobrescritas individuais de cores (valores hex) */
  colors?: Partial<{
    primary: string;         // Cor principal do texto
    secondary: string;       // Cor secundária do texto
    tertiary: string;        // Cor do texto abafado
    surfacePrimary: string;  // Cor de fundo primária
    surfaceSecondary: string; // Fundo secundário (cartões, painéis)
    brand: string;           // Cor de destaque/marca (botões, links)
    borderPrimary: string;   // Cor da borda primária
    borderSecondary: string; // Cor da borda secundária
  }>;
}
```

<Note>
  A prop `brandColor` é um atalho que define a cor `brand`. Se `brandColor` e `colors.brand` forem fornecidos, `colors.brand` tem precedência.
</Note>

### Exemplos de tema

<Tabs>
<Tab title="Apenas Cor da Marca">
```tsx
// Branding rápido — basta definir sua cor de marca
<AnySpend
  recipientAddress="0x..."
  theme={{ brandColor: "#6366f1" }}
/>
```
</Tab>

<Tab title="Tema Escuro">
```tsx

  recipientAddress="0x..."
  theme={{
    brandColor: "#10b981",
    colors: {
      primary: "#f9fafb",
      secondary: "#9ca3af",
      tertiary: "#6b7280",
      surfacePrimary: "#111827",
      surfaceSecondary: "#1f2937",
      borderPrimary: "#374151",
      borderSecondary: "#4b5563",
    },
  }}
/>
```
</Tab>

```tsx

  items={items}

      brand: "#e11d48",      // Acento Rosa
      surfaceSecondary: "#fef2f2", // Fundo rosa claro
    },
  }}
/>
```
</Tab>
