## Submissões de Artigos

Envie ideias de artigos para sites de conteúdo cripto. A IA valida seu tópico e gera títulos e descrições otimizados.

### Como Funciona

<Steps>
  <Step title="Enviar Tópico ou Tweet">
    Forneça uma URL de tweet, ideia de tópico ou título completo do artigo
  </Step>
  <Step title="Validação por IA (Grátis)">
    A IA pontua seu conteúdo (0-100) e gera sugestões, se necessário
  </Step>
  <Step title="Obter Aprovação">
    - **Pontuação ≥80:** Sua entrada é perfeita, aceita como está
    - **Pontuação 70-79:** Aprovado com sugestões melhoradas pela IA
    - **Pontuação &lt;70:** Rejeitado - tente um tópico diferente
  </Step>
  <Step title="Enviar & Pagar">
    Uma assinatura x402 para enviar sua solicitação de artigo
  </Step>
  <Step title="Publicação">
    O site publica seu artigo e você recebe a URL ao vivo
  </Step>
</Steps>

### Preços

Os preços variam por site, tipicamente **$3-50 USDC** por artigo. Cada site define sua própria taxa com base no tráfego e público.

<Note>
  A validação é grátis! Você pode validar quantas ideias quiser antes de se comprometer com o pagamento.
</Note>

### Comprar Espaço Publicitário (Pagamento x402)

```bash
POST https://x402.anyspend.com/ads/buy
```

Compre espaços publicitários programaticamente usando pagamentos sem gás do x402. Este endpoint requer autorização de pagamento x402.

**Corpo da Requisição:**

<CodeGroup>
```json Request
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Texto alternativo",
    "title": "Título do anúncio",
    "description": "Descrição"
  }
}
```

```json Resposta de Sucesso (200)
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json Resposta de Erro (400)
{
  "success": false,
  "error": "Espaço já reservado para este horário"
}
```
</CodeGroup>

**Parâmetros:**
- `productId` (obrigatório) - O ID do espaço publicitário de `/ads/slots`
- `buyerAddress` (obrigatório) - Seu endereço de carteira
- `startTime` (obrigatório) - Hora de início no formato ISO 8601
- `endTime` (obrigatório) - Hora de término no formato ISO 8601
- `adContent.imageUrl` (obrigatório) - URL direta para a imagem do anúncio
- `adContent.linkUrl` (obrigatório) - URL da página de destino
- `adContent.altText` (obrigatório) - Texto alternativo da imagem
- `adContent.title` (opcional) - Título do anúncio
- `adContent.description` (opcional) - Descrição do anúncio

**Exemplo em JavaScript/TypeScript:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// Criar cliente de carteira
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// Envolva o fetch com pagamento x402
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // Máximo de 1000 USDC
);

// Comprar espaço publicitário
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: 'Meu anúncio',
        title: 'Confira nosso produto!',
        description: 'Oferta limitada'
      }
    })
  }
);

const result = await response.json();
console.log('ID da Reserva:', result.data.bookingId);
```

<Note>
  Veja a [documentação do SDK x402](/anyspend/x402-overview) para mais detalhes sobre o uso de `wrapFetchWithPayment`.
</Note>

### Explorar Sites (Grátis)

```bash
GET https://x402.anyspend.com/ads/sites
```

Lista todos os sites disponíveis com paginação. Este endpoint é gratuito para chamar.

Parâmetros de consulta:
- `page` - Número da página (padrão: 1)
- `limit` - Resultados por página (padrão: 20, máximo: 100)
- `search` - Buscar por nome/domínio/palavras-chave

<CodeGroup>
```bash cURL
curl "https://x402.anyspend.com/ads/sites?search=defi&limit=10"
```


### Acompanhar Suas Campanhas (Grátis)

```bash
GET https://x402.anyspend.com/ads/bookings/buyer/:address
```

Acompanhe todas as suas reservas de anúncios - ativas, pendentes e concluídas. Este endpoint é gratuito para chamar.

<CodeGroup>
```bash cURL
curl "https://x402.anyspend.com/ads/bookings/buyer/0xYourAddress"
```

```json Resposta
{
  "success": true,
  "data": [
    {
      "bookingId": "booking_xyz789",
      "siteId": "abc123",
      "slotType": "banner",
      "startTime": "2024-02-01T10:00:00.000Z",
      "endTime": "2024-02-01T13:00:00.000Z",
      "adContent": {
        "imageUrl": "https://mysite.com/ad.png",
        "linkUrl": "https://mysite.com/landing"
      },
      "totalPrice": "3000000",
      "status": "active",
      "paymentTxHash": "0xabcdef...",
      "createdAt": "2024-01-22T16:30:00.000Z"
    }
  ]
}
```
</CodeGroup>

**Valores de status:**
- `active` - Em execução atualmente
- `pending` - Agendado para o futuro
- `completed` - Campanha encerrada

<Info>
  Os preços nas respostas da API estão em wei de USDC (6 decimais). Divida por 1.000.000 para obter o valor em USDC.
</Info>
