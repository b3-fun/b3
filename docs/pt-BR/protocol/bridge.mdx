## Ponte Nativa

<div style={{ marginBottom: '16px' }}>
  <span style={{ backgroundColor: '#3b82f6', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Base para B3</span>
  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Retiradas de 7 Dias</span>
  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>Sem Taxas</span>
</div>

Faça a ponte facilmente de Base para B3 usando a ponte canônica oficial da OP Stack!

<Warning>
Retiradas de B3 de volta para Base têm um período de desafio de 7 dias por segurança. Use pontes de terceiros para retiradas instantâneas.
</Warning>

<Steps>
<Step title="Visite a ponte">
  Acesse [bridge.b3.fun](https://bridge.b3.fun) para acessar a ponte oficial.
</Step>

<Step title="Conecte sua Carteira">
  Conecte sua carteira à interface da ponte.
</Step>

<Step title="Escolha a direção">
  Selecione se você deseja:
  - **Depositar**: Trazer fundos de Base → B3
  - **Retirar**: Transferir de volta de B3 → Base
</Step>

<Step title="Insira a quantidade">
  Insira a quantidade de ETH que deseja transferir.
</Step>

<Step title="Complete a transação">
  Clique em "Depositar" e confirme a transação na sua carteira.
  
  <Tip>
  Depósitos para B3 são instantâneos, mas retiradas levam 7 dias pelo período de desafio.
  </Tip>
</Step>
</Steps>

</Card>

## Transferência Nativa (Contrato da Ponte)

  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>Sem Taxas</span>
</div>

Envie ETH Base através do contrato da ponte. Transfira **ETH Base** para o seguinte endereço da ponte no **Mainnet Base**:

<Warning>
Este método é apenas para usuários avançados. Certifique-se de que está enviando do Mainnet Base para evitar a perda de fundos.
</Warning>

```text Endereço da Ponte
0x769547a723783FCA36BAaf1ECcf9dfdbF6d09F38
```

<Note>
Quando você enviar ETH para este endereço a partir de Base, ele aparecerá automaticamente na sua carteira em B3. Este é um método de interação direta com o contrato.
</Note>

<Accordion title="Como usar o contrato da ponte">
1. **Certifique-se de que está no Mainnet Base** na sua carteira
2. **Envie ETH** para o endereço do contrato da ponte acima
3. **Aguarde a confirmação** - seus fundos aparecerão em B3 automaticamente
4. **Verifique sua carteira B3** - mude para a rede B3 para ver seu ETH transferido

<Warning>
Verifique duas vezes se está no Mainnet Base antes de enviar. Enviar de outras redes pode resultar em perda de fundos.
</Warning>
</Accordion>
