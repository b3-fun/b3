## O que é o Basement?
Basement é a camada focada em jogos do ecossistema B3, projetada especificamente para desenvolvedores de jogos e comunidades de jogos. Ele fornece:

<img src="/images/bsmnt-logo.svg" alt="Basement.fun" className="w-32" />

- **Game Launcher**: Implante e gerencie jogos no B3
- **Ferramentas para Desenvolvedores**: SDKs e APIs para desenvolvimento de jogos
- **Recursos da Comunidade**: Perfis de jogadores, conquistas e recursos sociais
- **Recursos de Marca**: Ativos e diretrizes para uma marca consistente

## Principais Recursos

<AccordionGroup>
<Accordion title="SDKs para Desenvolvimento de Jogos">
Kits de desenvolvimento de software abrangentes para Unity, Construct e outros motores de jogos populares, permitindo integração perfeita com os recursos de blockchain do B3.
</Accordion>

<Accordion title="Identidade e Contas dos Jogadores">
Sistema global de contas que permite aos jogadores manter perfis, conquistas e ativos consistentes em todos os jogos do ecossistema Basement.
</Accordion>

<Accordion title="Gerenciamento de Ativos">
Ferramentas para gerenciar ativos dentro do jogo, NFTs e tokens, com fácil integração para desenvolvedores e propriedade transparente para os jogadores.
</Accordion>

<Accordion title="Recursos da Comunidade">
Recursos sociais integrados, classificações, torneios e ferramentas de gestão de comunidade para aumentar o engajamento dos jogadores.
</Accordion>
</AccordionGroup>
