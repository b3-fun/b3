## Classes Principais

### BondkitTokenFactory

A classe factory lida com a implantação de novos tokens de títulos e a consulta de tokens implantados.

```typescript
import { BondkitTokenFactory } from "@b3dotfun/sdk/bondkit";
import { base } from "viem/chains";

const factory = new BondkitTokenFactory(
  base.id,                    // ID da Cadeia (Base mainnet)
  process.env.WALLET_KEY      // Opcional: chave privada para escritas
);
```

#### Construtor

```typescript
constructor(
  chainId: SupportedChainId,
  walletKey?: string
)
```

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-------------|
| `chainId` | `number` | Sim | ID da Cadeia (atualmente apenas Base: 8453) |
| `walletKey` | `string` | Não | Chave privada para operações de escrita |

### BondkitToken

A classe token lida com todas as operações para um token de título específico.

```typescript
import { BondkitToken } from "@b3dotfun/sdk/bondkit";

const token = new BondkitToken(
  "0x123...",                 // Endereço do Token
  process.env.WALLET_KEY      // Opcional: chave privada
);
```

#### Construtor

```typescript
constructor(
  contractAddress: string,
  walletKey?: string
)
```

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-------------|
| `contractAddress` | `string` | Sim | Endereço do contrato do token |
| `walletKey` | `string` | Não | Chave privada para operações de escrita |

#### Métodos de Escrita

<AccordionGroup>
  <Accordion title="buy">
    **Comprar tokens com ETH**
    
    ```typescript
    async buy(
      minTokensOut: bigint,
      ethAmount: string | bigint
    ): Promise<Hex>
    ```
    
    **Parâmetros:**
    - `minTokensOut`: Mínimo de tokens a receber (proteção contra deslizamento)
    - `ethAmount`: ETH a gastar (como string ou bigint)
    
    **Exemplo:**
    ```typescript
    // Obter cotação primeiro
    const quote = await token.getAmountOfTokensToBuy(
      parseEther("1")
    );
    
    // Comprar com tolerância de deslizamento de 5%
    const minTokens = quote * 95n / 100n;
    const txHash = await token.buy(minTokens, "1");
    
    console.log(`Transação: ${txHash}`);
    ```
  </Accordion>
  
  <Accordion title="sell">
    **Vender tokens por ETH**
    
    ```typescript
    async sell(
      tokenAmount: bigint,
      minEthOut: bigint
    ): Promise<Hex>
    ```
    
    **Parâmetros:**
    - `tokenAmount`: Quantidade de tokens a vender
    - `minEthOut`: Mínimo de ETH a receber (proteção contra deslizamento)
    
    **Exemplo:**
    ```typescript
    // Vender 1000 tokens
    const sellAmount = parseEther("1000");
    
    // Obter cotação
    const quote = await token.getAmountOfEthToReceive(sellAmount);
    
    // Vender com proteção contra deslizamento
    const minEth = quote * 95n / 100n;
    const txHash = await token.sell(sellAmount, minEth);
    ```
  </Accordion>
  
  <Accordion title="migrateToDex">
    **Migrar para Uniswap v4**
    
    ```typescript
    async migrateToDex(): Promise<Hex>
    ```
    
    **Requisitos:**
    - O chamador deve ser o administrador da migração
    - O alvo deve ser alcançado
    
    **Exemplo:**
    ```typescript
    if (await token.canMigrate()) {
      const txHash = await token.migrateToDex();
      console.log(`Tx de migração: ${txHash}`);
    }
    ```
  </Accordion>

```typescript

## Tratamento de Erros

```typescript
try {
  const txHash = await token.buy(minTokens, "1");
} catch (error) {
  if (error.message.includes("insufficient funds")) {
    console.error("ETH insuficiente");
  } else if (error.message.includes("slippage")) {
    console.error("Preço movido, tente novamente");
  } else if (error.message.includes("target exceeded")) {
    console.error("Ultrapassaria o alvo de migração");
  } else {
    console.error("Transação falhou:", error);
  }
}
```

## Exemplos Completos

### Implantar e Negociar

```typescript
import { 
  BondkitTokenFactory, 
  BondkitToken 
} from "@b3dotfun/sdk/bondkit";
import { parseEther, formatEther } from "viem";
import { base } from "viem/chains";

async function deployAndTrade() {
  // 1. Implantar token
  const factory = new BondkitTokenFactory(
    base.id,
    process.env.WALLET_KEY
  );
  
  const tokenAddress = await factory.deployBondkitToken({
    name: "Test Token",
    symbol: "TEST",
    finalTokenSupply: parseEther("1000000"),
    aggressivenessFactor: 50,
    targetEth: parseEther("10"),
    feeRecipient: process.env.WALLET_ADDRESS,
    lpSplitRatioFeeRecipientBps: 1000n,
    migrationAdminAddress: process.env.WALLET_ADDRESS,
    uniswapV2RouterAddress: "0x4752ba5DBc23f44D87826276BF6Fd6b1C372aD24"
  });
  
  // 2. Inicializar cliente do token
  const token = new BondkitToken(
    tokenAddress,
    process.env.WALLET_KEY
  );
  
  // 3. Comprar tokens
  const buyAmount = parseEther("0.5");
  const expectedTokens = await token.getAmountOfTokensToBuy(buyAmount);
  await token.buy(expectedTokens * 95n / 100n, "0.5");
