## Pengajuan Artikel

Kirim ide artikel ke situs konten kripto. AI memvalidasi topik Anda dan menghasilkan judul dan deskripsi yang dioptimalkan.

### Cara Kerja

<Steps>
  <Step title="Kirim Topik atau Tweet">
    Berikan URL tweet, ide topik, atau judul artikel lengkap
  </Step>
  <Step title="Validasi AI (Gratis)">
    AI menilai konten Anda (0-100) dan menghasilkan saran jika diperlukan
  </Step>
  <Step title="Dapatkan Persetujuan">
    - **Skor ≥80:** Input Anda sempurna, diterima apa adanya
    - **Skor 70-79:** Disetujui dengan saran yang diperbaiki AI
    - **Skor &lt;70:** Ditolak - coba topik lain
  </Step>
  <Step title="Kirim & Bayar">
    Satu tanda tangan x402 untuk mengirim permintaan artikel Anda
  </Step>
  <Step title="Publikasi">
    Situs mempublikasikan artikel Anda dan Anda menerima URL langsung
  </Step>
</Steps>

### Harga

Harga bervariasi berdasarkan situs, biasanya **$3-50 USDC** per artikel. Setiap situs menetapkan tarif mereka sendiri berdasarkan lalu lintas dan audiens.

<Note>
  Validasi gratis! Anda dapat memvalidasi sebanyak mungkin ide sebelum berkomitmen untuk pembayaran.
</Note>

### Pembelian Slot Iklan (Pembayaran x402)

```bash
POST https://x402.anyspend.com/ads/buy
```

Pembelian slot iklan secara programatis menggunakan pembayaran tanpa gas x402. Endpoint ini memerlukan otorisasi pembayaran x402.

**Isi Permintaan:**

<CodeGroup>
```json Permintaan
{
  "productId": "slot_abc123",
  "buyerAddress": "0x742d35Cc6...",
  "startTime": "2025-01-15T14:00:00Z",
  "endTime": "2025-01-15T18:00:00Z",
  "adContent": {
    "imageUrl": "https://...",
    "linkUrl": "https://...",
    "altText": "Teks alternatif",
    "title": "Judul iklan",
    "description": "Deskripsi"
  }
}
```

```json Respon Sukses (200)
{
  "success": true,
  "data": {
    "purchaseId": "purch_123...",
    "bookingId": "booking_456...",
    "totalPrice": "5000000",
    "status": "confirmed"
  }
}
```

```json Respon Error (400)
{
  "success": false,
  "error": "Slot sudah dipesan pada waktu ini"
}
```
</CodeGroup>

**Parameter:**
- `productId` (wajib) - ID slot iklan dari `/ads/slots`
- `buyerAddress` (wajib) - Alamat dompet Anda
- `startTime` (wajib) - Waktu mulai dalam format ISO 8601
- `endTime` (wajib) - Waktu berakhir dalam format ISO 8601
- `adContent.imageUrl` (wajib) - URL langsung ke gambar iklan
- `adContent.linkUrl` (wajib) - URL halaman arahan
- `adContent.altText` (wajib) - Teks alternatif gambar
- `adContent.title` (opsional) - Judul iklan
- `adContent.description` (opsional) - Deskripsi iklan

**Contoh JavaScript/TypeScript:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// Membuat klien dompet
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

// Membungkus fetch dengan pembayaran x402
const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // Maks 1000 USDC
);

// Membeli slot iklan
const response = await fetchWithPayment(
  'https://x402.anyspend.com/ads/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      productId: 'slot_abc123',
      buyerAddress: '0x742d35Cc6...',
      startTime: '2025-01-15T14:00:00Z',
      endTime: '2025-01-15T18:00:00Z',
      adContent: {
        imageUrl: 'https://example.com/ad.jpg',
        linkUrl: 'https://example.com',
        altText: 'Iklan saya',
        title: 'Lihat produk kami!',
        description: 'Penawaran terbatas'
      }
    })
  }
);

const result = await response.json();
console.log('ID Pemesanan:', result.data.bookingId);
```

<Note>
  Lihat [dokumentasi SDK x402](/anyspend/x402-overview) untuk lebih detail tentang menggunakan `wrapFetchWithPayment`.
</Note>

### Jelajahi Situs (Gratis)

```bash
GET https://x402.anyspend.com/ads/sites
```

Daftar semua situs yang tersedia dengan pagination. Endpoint ini gratis untuk dipanggil.

Parameter query:
- `page` - Nomor halaman (default: 1)
- `limit` - Hasil per halaman (default: 20, maks: 100)
- `search` - Cari berdasarkan nama/domain/kata kunci

<CodeGroup>
```bash cURL
curl "https://x402.anyspend.com/ads/sites?search=defi&limit=10"
```

```javascript JavaScript

  .then(r => r.json());
```

### Validasi Ide Artikel (Gratis)

```bash
POST https://x402.anyspend.com/articles/validate
```

Validasi ide artikel dan dapatkan judul dan deskripsi yang dihasilkan AI. Endpoint ini gratis untuk dipanggil - tidak memerlukan pembayaran.

**Isi Permintaan:**

<CodeGroup>
```json Permintaan
{
  "siteDomain": "baseradar.com",
  "userInput": "Cara Menggunakan Base Pay untuk Kreator di DApp Anda"
}
```

```json Respon Disetujui (Skor ≥70)
{
  "success": true,
  "data": {
    "approved": true,
    "score": 92,
    "reasoning": "Cocok sempurna untuk konten ekosistem Base",
    "estimatedPrice": "15000000",
    "suggestedTitle": "Cara Menggunakan Base Pay untuk Kreator di DApp Anda",
    "suggestedDescription": "Pelajari cara mengintegrasikan Base Pay...",
    "userProvidedTitle": true,
    "message": "Input Anda disetujui apa adanya"
  }
}
```

```json Respon Ditolak (Skor <70)
{
  "success": false,
  "data": {
    "approved": false,
    "score": 45,
    "reasoning": "Topik tidak sejalan dengan fokus situs pada ekosistem Base",
    "estimatedPrice": "15000000",
    "message": "Konten tidak sesuai fokus situs"
  }
}
```

**Parameter:**
- `siteDomain` (wajib) - Domain situs target (misal, "baseradar.com", "solanaecho.com
