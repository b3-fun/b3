## Slots

Slots memungkinkan Anda menggantikan bagian UI secara keseluruhan dengan komponen React Anda sendiri. Setiap slot menerima properti dari AnySpend yang dapat Anda gunakan dalam implementasi kustom Anda.

```typescript title="AnySpendSlots Interface" icon="code"
interface AnySpendSlots {
  /** Gantikan tombol aksi/pembayaran utama */
  actionButton?: (props: ActionButtonSlotProps) => ReactNode;
  /** Gantikan tombol sambungkan dompet */
  connectWalletButton?: (props: ConnectWalletButtonSlotProps) => ReactNode;
  /** Gantikan header */
  header?: (props: { mode: "page" | "modal" }) => ReactNode;
  /** Gantikan footer */
  footer?: ReactNode;
  /** Gantikan layar sukses */
  successScreen?: (props: SuccessScreenSlotProps) => ReactNode;
  /** Gantikan layar error */
  errorScreen?: (props: ErrorScreenSlotProps) => ReactNode;
  /** Gantikan panel formulir checkout (hanya checkout) */
  checkoutForm?: (props: CheckoutFormComponentProps) => ReactNode;
  /** Gantikan pemilih metode pengiriman (hanya checkout) */
  shippingSelector?: (props: ShippingSelectorSlotProps) => ReactNode;
  /** Gantikan input kode diskon (hanya checkout) */
  discountInput?: (props: DiscountInputSlotProps) => ReactNode;
}
```

<Tip>
  Slot `checkoutForm`, `shippingSelector`, dan `discountInput` hanya digunakan di `<AnySpendCheckout>` dan `<AnySpendCheckoutTrigger>`. Lihat [Panduan Checkout](/anyspend/checkout) untuk detail tentang fitur formulir/pengiriman/diskon.
</Tip>

## Theme

Konfigurasikan tampilan visual komponen AnySpend dengan warna merek.

```typescript title="AnySpendTheme Interface" icon="code"
interface AnySpendTheme {
  /** Warna merek utama (hex). Diterapkan sebagai warna aksen utama. */
  brandColor?: string;
  /** Penggantian warna individu (nilai hex) */
  colors?: Partial<{
    primary: string;         // Warna teks utama
    secondary: string;       // Warna teks sekunder
    tertiary: string;        // Warna teks redup
    surfacePrimary: string;  // Warna latar belakang utama
    surfaceSecondary: string; // Latar belakang sekunder (kartu, panel)
    brand: string;           // Warna aksen/merek (tombol, tautan)
    borderPrimary: string;   // Warna batas utama
    borderSecondary: string; // Warna batas sekunder
  }>;
}
```

<Note>
  Properti `brandColor` adalah pintasan yang mengatur warna `brand`. Jika `brandColor` dan `colors.brand` keduanya disediakan, `colors.brand` memiliki prioritas.
</Note>
