## Ikhtisar

AnySpend x402 mendukung pembayaran lintas berbagai jaringan blockchain termasuk rantai yang kompatibel dengan EVM dan Solana (SVM). Halaman ini menyediakan referensi lengkap dari jaringan yang didukung, token, dan alamat kontrak mereka.

AnySpend x402 mendukung rantai mainnet EVM berikut:

|---------|-----------|----------|------|
| Abstract | `abstract` | 2741 | L2 |
| Arbitrum One | `arbitrum` | 42161 | L2 - Optimistic Rollup |
| Avalanche C-Chain | `avalanche` | 43114 | L1 |
| B3 | `b3` | 8333 | L2 |

## Token yang Didukung

### USDC (Circle)

| Jaringan | Alamat Kontrak | Desimal | Metode Tanda Tangan |
|---------|-----------------|----------|------------------|
| Base | `0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913` | 6 | transferWithAuthorization |
| Ethereum | `0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48` | 6 | transferWithAuthorization |
| Arbitrum | `0xaf88d065e77c8cC2239327C5EDb3A432268e5831` | 6 | transferWithAuthorization |
| Optimism | `0x0b2C639c533813f4Aa9D7837CAf62653d097Ff85` | 6 | transferWithAuthorization |
| Polygon | `0x3c499c542cEF5E3811e1192ce70d8cC03d5c3359` | 6 | transferWithAuthorization |
| Base Sepolia | `0x036CbD53842c5426634e7929541eC2318f3dCF7e` | 6 | transferWithAuthorization |

| Jaringan | Alamat Kontrak | Desimal | Metode Tanda Tangan |
|---------|-----------------|----------|------------------|
| Base | `0x50c5725949A6F0c72E6C4a641F24049A917DB0Cb` | 18 | permit (EIP-2612) |
| Ethereum | `0x6B175474E89094C44Da98b954EedeAC495271d0F` | 18 | permit (EIP-2612) |
| Arbitrum | `0xDA10009cBd5D07dd0CeCc66161FC93D7c9000da1` | 18 | permit (EIP-2612) |
| Optimism | `0xDA10009cBd5D07dd0CeCc66161FC93D7c9000da1` | 18 | permit (EIP-2612) |
| Polygon | `0x8f3Cf7ad23Cd3CaDbD9735AFf958023239c6A063` | 18 | permit (EIP-2612) |

**Penggunaan:**
```typescript

```

### USDT (Tether)

Dukungan USDT bervariasi berdasarkan jaringan. Beberapa implementasi mendukung permit, yang lain tidak.

| Jaringan | Alamat Kontrak | Desimal | Metode Tanda Tangan |
|---------|-----------------|----------|------------------|
| Base | `0xfde4C96c8593536E31F229EA8f37b2ADa2699bb2` | 6 | permit (EIP-2612) |
| Ethereum | `0xdAC17F958D2ee523a2206206994597C13D831ec7` | 6 | ⚠️ Tidak mendukung permit |
| Arbitrum | `0xFd086bC7CD5C481DCC9C85ebE478A1C0b69FCbb9` | 6 | permit (EIP-2612) |
| Optimism | `0x94b008aA00579c1307B0EF2c499aD98a8ce58e58` | 6 | permit (EIP-2612) |
| Polygon | `0xc2132D05D31c914a87C6611C10748AEb04B58e8F` | 6 | ⚠️ Tidak mendukung permit |

<Warning>
USDT di Ethereum dan Polygon **tidak** mendukung EIP-2612 permit. Gunakan alur `approve()` + `transferFrom()` tradisional atau pilih token lain.
</Warning>

### Token ERC-20 Lainnya

Setiap token ERC-20 yang mengimplementasikan **EIP-2612 permit** didukung:

- **cbBTC** (Coinbase Wrapped Bitcoin)
- **LINK** (Chainlink)
- **UNI** (Uniswap)
- **AAVE** (Aave)
- Dan banyak lagi...

**Periksa apakah token mendukung permit:**
```typescript
// Periksa apakah token memiliki fungsi permit
const hasPermit = await token.read.permit !== undefined;

// Atau periksa untuk DOMAIN_SEPARATOR (diperlukan untuk EIP-712)
const hasDomainSeparator = await token.read.DOMAIN_SEPARATOR !== undefined;
```

<Info>
Untuk informasi terperinci tentang tanda tangan tanpa gas, lihat panduan [Permit vs TransferWithAuthorization](/anyspend/x402-signature-methods).
</Info>

## Konfigurasi Token

### Sisi Klien (Membayar)

Tentukan token pembayaran pilihan Anda:

```typescript
import { X402Client } from 'anyspend-x402-client';

const client = new X402Client({
  walletClient,
  preferredToken: '0x50c5725949A6F0c72E6C4a641F24049A917DB0Cb', // DAI di Base
  network: 'base-mainnet'
});
```

Atau per-permintaan:

```typescript
const response = await client.request('https://api.example.com/data', {
  preferredToken: '0x50c5725949A6F0c72E6C4a641F24049A917DB0Cb' // DAI di Base
});
```

### Sisi Server (Menerima)

Konfigurasikan token yang diterima dan mata uang penyelesaian:

```typescript
import { paymentMiddleware, facilitator } from '@b3dotfun/anyspend-x402';

const endpointConfig = {
  '/api/data': {
    amount: '1.00',        // $1 nilai dari token apa pun
    asset: 'USD',          // Biarkan klien memilih token
    network: 'base-mainnet'
  }
};

app.use(paymentMiddleware(
  '0xYourUSDCAddress...',  // Terima USDC
  endpointConfig,
  facilitator
));
```

## Dukungan Lintas-Rantai

AnySpend dapat memfasilitasi pembayaran di mana **token klien** berada di rantai yang berbeda dari **penyelesaian server sumber**.

**Contoh:**
```typescript
// Klien membayar dengan DAI di Ethereum
const clientPayment = {
  token: '0x6B175474E89094C44Da98b954EedeAC495271d0F', // DAI di Ethereum
  network: 'ethereum-mainnet',
  amount: '1.0'
};

// Server sumber menerima USDC di Base
const serverSettlement = {
  token: '0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913', // USDC di Base
  network: 'base-mainnet',
  amount: '1.00'
};

// AnySpend menangani:
// 1. Terima DAI di Ethereum
// 2. Jembatani DAI dari Ethereum → Base
// 3. Tukar DAI → USDC di Base
// 4. Penyelesaian USDC ke server sumber
```

**Rute Lintas-Rantai yang Didukung:**
AnySpend mendukung rute lintas-rantai antara semua jaringan EVM yang didukung, termasuk:
- Ethereum ↔ Base, Arbitrum, Optimism, Polygon, Avalanche
- Base ↔ Arbitrum, Optimism, Polygon
- L1 ↔ L2 bridging lintas semua rantai yang didukung
- Lintas-VM: Solana ↔ rantai EVM (eksperimental)

<Info>
Pembayaran lintas-rantai membutuhkan waktu lebih lama (2-5 menit) karena waktu bridging dan memerlukan biaya tambahan (~0,5% total). Pembayaran lintas-rantai disarankan untuk pengalaman pengguna terbaik.
</Info>

| B3 | ~$0.001 | ~$0.003 | ~$0.01 - $0.05 |
| Arbitrum | ~$0.01 | ~$0.05 | ~$0.1 - $1 |
| Optimism | ~$0.01 | ~$0.05 | ~$0.1 - $1 |

</Note>

## Pengujian & Pengembangan

AnySpend mendukung beberapa testnet untuk pengembangan dan pengujian. Lihat bagian [Jaringan EVM Testnet](#evm-testnet-networks) di atas untuk daftar lengkap testnet yang didukung dan faucet mereka.

### Testnet yang Direkomendasikan

**Untuk pengembangan umum:**
- **Base Sepolia** (Chain ID: 84532) - Pengalaman testnet terbaik, cepat dan andal
- **Solana Devnet** (Chain ID: 103) - Untuk pengujian Solana/SVM

**Untuk pengujian jaringan spesifik:**
- Gunakan testnet yang sesuai dengan target mainnet Anda
- Lihat tabel Jaringan EVM Testnet di atas untuk tautan faucet

### Alamat Token Uji

Hubungi tim AnySpend untuk alamat token uji di jaringan tertentu, atau gunakan faucet testnet standar untuk mendapatkan token asli untuk pengujian.
