## Apa itu AnySpend x402?

AnySpend x402 adalah implementasi B3 dari **protokol pembayaran x402** yang memungkinkan pembayaran kriptokurensi secara instan dan otomatis langsung melalui HTTP. Tidak seperti fasilitator x402 tradisional yang hanya mendukung USDC, **AnySpend memungkinkan Anda membayar dengan token apa pun** sementara server sumber menerima token apa pun yang mereka inginkan.

Protokol x402 memodernisasi kode status HTTP `402 Payment Required`, mengubahnya menjadi solusi praktis untuk mikropembayaran mesin-ke-mesin, API berbayar, dan interaksi agen otonom.

## Untuk Pembeli (Pembayar)

Bayar untuk sumber daya berbayar menggunakan kriptokurensi apa pun yang Anda miliki:

- **Bayar dengan token apa pun**: Gunakan ETH, DAI, atau token ERC-20 apa pun - tidak hanya USDC
- **Protokol standar**: Bekerja dengan layanan yang diaktifkan x402 apa pun menggunakan HTTP standar
- **Tanpa penukaran manual**: AnySpend secara otomatis mengonversi token Anda ke apa yang dibutuhkan penjual
- **Integrasi sederhana**: Satu SDK bekerja di semua layanan x402

**Kasus Penggunaan:**
- Akses API premium dan layanan agen AI
- Bayar untuk sumber daya komputasi dan umpan data
- Aktifkan pembayaran agen-ke-agen otonom
- Konsumsi konten berbayar secara programatis

## Untuk Penjual (Server Sumber)

Terima pembayaran dalam USDC sementara pelanggan Anda membayar dengan token pilihan mereka:

- **Terima USDC**: Dibayar dalam USDC terlepas dari token apa yang digunakan pelanggan
- **Middleware Express**: Tambahkan pembayaran ke API Anda hanya dengan beberapa baris kode
- **Tanpa infrastruktur blockchain**: AnySpend menangani semua kompleksitas on-chain
- **Penyelesaian instan**: Pembayaran diverifikasi dan diselesaikan secara otomatis

**Kasus Penggunaan:**
- Monetisasi API premium dan layanan data
- Bangun alur kerja agen AI berbayar
- Buat konten dan alat berbayar
- Terima pembayaran kripto tanpa keahlian kripto

## Pembeda Utama

### Dukungan Multi-Token

**x402 Standar**: Server sumber dan klien harus sama-sama menggunakan USDC

**AnySpend x402**: Klien membayar dengan **token apa pun** yang mereka pegang. AnySpend menangani konversinya.

```
Klien membayar 0.0003 ETH → AnySpend menukar ke 1 USDC → Penjual menerima 1 USDC
```

### Tidak Memerlukan Infrastruktur

Server sumber tidak perlu:
- Menjalankan node blockchain
- Mengelola dompet atau kunci pribadi
- Menangani penukaran token atau bridging
- Memantau status transaksi

Fasilitator AnySpend menangani semua operasi blockchain dan menyediakan titik akhir HTTP sederhana untuk verifikasi dan penyelesaian.

### Kepatuhan Protokol Standar

Klien yang menggunakan AnySpend memiliki **nol kesadaran** mereka menggunakan fasilitator khusus. Seluruh alur menggunakan pesan protokol x402 standar - keajaiban multi-token terjadi secara transparan di sisi server sumber melalui middleware.

- ✅ Penukaran token otomatis dan routing lintas rantai
- ✅ Pembayaran tanpa gas untuk pengguna

- ✅ Pengganti langsung untuk fasilitator x402 apa pun
