## Apa itu AnySpend x402

AnySpend x402 adalah implementasi B3 dari **protokol pembayaran x402** yang memungkinkan pembayaran kriptokurensi instan dan otomatis langsung melalui HTTP. Berbeda dengan fasilitator x402 tradisional yang hanya mendukung USDC, **AnySpend memungkinkan Anda membayar dengan token yang kompatibel apa pun** (EIP-2612 atau EIP-3009) sementara server sumber menerima token apa pun yang mereka inginkan.

Protokol x402 memodernisasi kode status HTTP `402 Payment Required`, mengubahnya menjadi solusi praktis untuk mikropembayaran antar mesin, API berbayar, dan interaksi agen otonom.

## Untuk pembeli (pembayar)

Bayar untuk sumber daya berbayar menggunakan token yang kompatibel apa pun yang Anda miliki:


- **Bayar dengan token yang kompatibel**: Gunakan token apa pun dengan dukungan EIP-2612 atau EIP-3009 untuk pembayaran tanpa gas
- **Protokol standar**: Bekerja dengan layanan yang diaktifkan x402 apa pun menggunakan HTTP standar
- **Tidak perlu swap manual**: AnySpend secara otomatis mengonversi token Anda ke apa yang dibutuhkan penjual
- **Integrasi sederhana**: Satu SDK bekerja di semua layanan x402

**Kasus Penggunaan:**
- Akses API premium dan layanan agen AI
- Bayar untuk sumber daya komputasi dan umpan data
- Aktifkan pembayaran agen-ke-agen otonom
- Konsumsi konten berbayar secara programatis

## Untuk penjual (server sumber)

Terima pembayaran dalam USDC sementara pelanggan Anda membayar dengan token pilihan mereka:

- **Terima USDC**: Dibayar dalam USDC terlepas dari token apa yang digunakan pelanggan
- **Middleware Express**: Tambahkan pembayaran ke API Anda hanya dengan beberapa baris kode
- **Tidak perlu infrastruktur blockchain**: AnySpend menangani semua kompleksitas on-chain
- **Penyelesaian instan**: Pembayaran diverifikasi dan diselesaikan secara otomatis

**Kasus Penggunaan:**
- Monetisasi API premium dan layanan data
- Bangun alur kerja agen AI berbayar
- Buat konten dan alat berbayar
- Terima pembayaran kripto tanpa keahlian kripto

## Pembeda utama

### Dukungan multi-token

**x402 standar**: Server sumber dan klien harus sama-sama menggunakan USDC

**AnySpend x402**: Klien membayar dengan **token yang kompatibel apa pun** (dukungan EIP-2612 atau EIP-3009 diperlukan) yang mereka miliki. AnySpend menangani konversinya.

```
Klien membayar dengan DAI (EIP-2612) → AnySpend menukar ke 1 USDC → Penjual menerima 1 USDC
```

### Tidak memerlukan infrastruktur

Server sumber tidak perlu:
- Menjalankan node blockchain
- Mengelola dompet atau kunci pribadi
- Menangani swap token atau bridging
- Memantau status transaksi

Fasilitator AnySpend menangani semua operasi blockchain dan menyediakan endpoint HTTP sederhana untuk verifikasi dan penyelesaian.

### Kepatuhan protokol standar

Klien yang menggunakan AnySpend tidak perlu tahu bahwa mereka menggunakan fasilitator khusus. Alur menggunakan pesan protokol x402 standar, dan konversi multi-token terjadi di sisi server sumber melalui middleware.

## Didukung oleh fasilitator AnySpend

Semua pembayaran x402 diproses melalui Fasilitator AnySpend, yang menangani:

- Dukungan multi-token di lebih dari 19 jaringan blockchain
- Swap token otomatis dan routing lintas rantai
- Pembayaran tanpa gas untuk pengguna
- Implementasi x402 yang sesuai standar
- Pengganti langsung untuk fasilitator x402 apa pun
