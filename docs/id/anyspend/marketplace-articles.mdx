### Mengajukan Permintaan Artikel (Pembayaran x402)

```bash
POST https://x402.anyspend.com/articles/buy
```

Kirim permintaan artikel dengan pembayaran x402. Anda harus memvalidasi konten Anda terlebih dahulu dan menerima persetujuan.

**Body Permintaan:**

<CodeGroup>
```json Request
{
  "buyerAddress": "0x742d35Cc6...",
  "siteDomain": "baseradar.com",
  "title": "Mengintegrasikan Base Pay: Panduan Lengkap untuk Kreator Konten",
  "description": "Pelajari cara mengimplementasikan Base Pay di platform kreator Anda dengan contoh langkah demi langkah dan praktik terbaik."
}
```

```json Respons Sukses (200)
{
  "success": true,
  "data": {
    "requestId": "req_789...",
    "productSpecificId": "req_789...",
    "totalPrice": "50000000",
    "status": "pending"
  }
}
```

```json Respons Kesalahan (400)
{
  "success": false,
  "error": "Validasi konten diperlukan sebelum pengajuan"
}
```
</CodeGroup>

**Parameter:**
- `buyerAddress` (diperlukan) - Alamat wallet Anda
- `siteDomain` (diperlukan) - Domain situs target
- `title` (diperlukan) - Judul artikel (dari validasi atau milik Anda sendiri jika skor â‰¥90)
- `description` (diperlukan) - Deskripsi artikel (dari validasi atau milik Anda sendiri)

**Contoh JavaScript/TypeScript:**

```typescript
import { wrapFetchWithPayment } from '@b3dotfun/anyspend-x402-fetch';
import { createWalletClient, custom, publicActions } from 'viem';
import { base } from 'viem/chains';

// Langkah 1: Validasi konten Anda (GRATIS)
const validateResponse = await fetch('https://x402.anyspend.com/articles/validate', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    siteDomain: 'baseradar.com',
    userInput: 'Cara menggunakan Base Pay untuk kreator'
  })
});

const validation = await validateResponse.json();

if (!validation.data.approved) {
  console.error('Konten ditolak:', validation.data.reasoning);
  process.exit(1);
}

console.log('âœ… Disetujui! Skor:', validation.data.score);
console.log('Judul:', validation.data.suggestedTitle);

// Langkah 2: Kirim dengan pembayaran (x402)
const walletClient = createWalletClient({
  account: '0x742d35Cc6...',
  chain: base,
  transport: custom(window.ethereum),
}).extend(publicActions);

const fetchWithPayment = wrapFetchWithPayment(
  fetch,
  walletClient,
  BigInt(1000 * 10 ** 6) // Maks 1000 USDC
);

const response = await fetchWithPayment(
  'https://x402.anyspend.com/articles/buy',
  {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      buyerAddress: '0x742d35Cc6...',
      siteDomain: 'baseradar.com',
      title: validation.data.suggestedTitle,
      description: validation.data.suggestedDescription
    })
  }
);

const result = await response.json();
console.log('ðŸŽ‰ Artikel diajukan! ID Permintaan:', result.data.requestId);
```

<Note>
  Lihat [dokumentasi SDK x402](/anyspend/x402-overview) untuk detail lebih lanjut tentang penggunaan `wrapFetchWithPayment`.
</Note>

**Nilai status:**
- `pending` - Diajukan ke penerbit, menunggu tinjauan
- `accepted` - Penerbit menerima, artikel dalam produksi
- `published` - Artikel sudah live (termasuk `publishedUrl`)
- `rejected` - Penerbit menolak (jarang, biasanya sudah divalidasi sebelumnya)
