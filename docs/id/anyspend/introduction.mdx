## Apa itu AnySpend?

AnySpend adalah infrastruktur pembayaran komprehensif B3 yang memungkinkan transaksi kripto yang mulus lintas berbagai blockchain. Baik Anda sedang membangun aplikasi DeFi, pasar NFT, atau platform game, AnySpend menyediakan komponen dan API yang Anda butuhkan untuk menerima pembayaran dalam token apa pun.

[Coba AnySpend di browser Anda](https://anyspend.com/swap)

## Fitur

<Steps>
<Step title="Pengguna Memulai Pembayaran">
Pengguna mengklik tombol pembayaran atau berinteraksi dengan antarmuka Anda, memilih token yang mereka inginkan
</Step>

<Step title="AnySpend Memberikan Kutipan">
Mesin routing kami menemukan jalur terbaik untuk mengonversi token mereka ke token tujuan yang dibutuhkan
</Step>

<Step title="Eksekusi Cross-Chain">
Jika diperlukan, token dijembatani lintas chain menggunakan infrastruktur yang aman dan teruji
</Step>

<Step title="Pengiriman Akhir">
Token tiba di alamat tujuan, mengeksekusi logika kustom apa pun, dan memicu callback sukses Anda
</Step>
</Steps>

## Mulai Cepat

Mulai dengan AnySpend hanya dalam beberapa langkah:

```tsx
import { AnySpend, AnyspendProvider } from "@b3dotfun/sdk/anyspend/react";
import { B3Provider } from "@b3dotfun/sdk/global-account/react";

import "@b3dotfun/sdk/index.css";

function App() {
  return (
    <B3Provider theme="light" environment="production">
      <AnyspendProvider>
        <AnySpend />
      </AnyspendProvider>
    </B3Provider>
  );
}

export default App
```
