### Membuat Akun Global B3 Anda

<Steps>
  <Step title="Kunjungi Explorer">
    Kunjungi https://explorer.b3.fun/.
  </Step>
  <Step title="Buka Sidebar">
    Di sisi kiri halaman, arahkan kursor untuk memperluas sidebar. Pilih opsi apa saja (seperti **Transaksi** atau **Jelajahi Aplikasi**) untuk menampilkan opsi masuk.

    ![Screenshot2025 09 24at6 23 08PM Pn](/docs/images/Screenshot2025-09-24at6.23.08PM.png)
  </Step>
  <Step title="Hubungkan Akun Anda">
    Di pojok kanan atas layar, klik **Connect**.

    ![Screenshot2025 09 24at6 26 24PM Pn](/docs/images/Screenshot2025-09-24at6.26.24PM.png)
  </Step>
  <Step title="Pilih Metode Login">
    - **Wallet (MetaMask, WalletConnect, dll.)** - Setujui permintaan sambungan di wallet Anda.
    - **Sosial / Email / Telepon** - Ikuti proses verifikasi yang sesuai untuk metode login Anda (misalnya, konfirmasi email, kode SMS, atau otorisasi login sosial).

    <img
      src="/docs/images/Screenshot2025-09-24at6.28.03PM.png"
      alt="Screenshot2025 09 24at6 28 03PM Pn"
      title="Screenshot2025 09 24at6 28 03PM Pn"
      className="mx-auto"
      style={{ width:"61%" }}
    />
  </Step>
</Steps>

Setelah diverifikasi, Akun Global B3 Anda akan dibuat dan Anda akan masuk.

### FAQ

<Accordion title="Apakah saya memerlukan wallet untuk membuat Akun Global B3?">
  Tidak. Anda dapat mendaftar dengan wallet, email, telepon, atau login sosial. Jika Anda memilih untuk masuk tanpa wallet, sebuah wallet terintegrasi akan secara otomatis dibuat untuk Anda di latar belakang. Ini memastikan Anda masih akan memiliki wallet yang terhubung ke akun Anda, meskipun Anda tidak membawa wallet Anda sendiri.
</Accordion>
