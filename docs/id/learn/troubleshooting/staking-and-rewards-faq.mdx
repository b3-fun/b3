## Pertanyaan yang Sering Diajukan

<AccordionGroup>
<Accordion title="Bagaimana cara staking bekerja?">
Staking token B3 Anda memberikan Anda hadiah pasif dan akses ke permainan Stake to Win. Untuk informasi lebih lanjut, silakan baca artikel kami **Apa itu Staking dan Bagaimana Cara Kerjanya**.
</Accordion>

<Accordion title="Berapa periode pendinginan ketika saya unstake?">
Setelah Anda unstake, ada periode pendinginan 45 hari sebelum Anda dapat menarik token Anda.
</Accordion>

<Accordion title="Saya sudah staking tapi tidak melihat hadiah. Apa yang harus saya lakukan?">
Tunggu beberapa menit dan muat ulang. Jika masih tidak muncul, konfirmasi transaksi Anda berhasil di blockchain. Jika transaksi Anda berhasil, tapi Anda masih tidak mengumpulkan token, hubungi dukungan.
</Accordion>

<Accordion title="Saya sudah unstake token saya tapi saya tidak melihatnya di dompet saya. Di mana uang saya?">
Unstake memicu periode pendinginan 45 hari. Selama waktu ini, token Anda masih ditahan di kolam staking dan belum tersedia di dompet Anda. Setelah periode pendinginan berakhir, Anda perlu kembali ke halaman staking dan klik **Tarik** untuk memindahkan token Anda kembali ke dompet Anda.
</Accordion>

<Accordion title="Bagaimana cara saya mengklaim hadiah staking saya?">
Hadiah staking saat ini dalam fase akumulasi dan belum bisa diklaim. Ketika klaim menjadi tersedia, kami akan memberitahu semua staker dan memperbarui halaman ini dengan langkah-langkah untuk mengklaim.
</Accordion>

<Accordion title="Apakah ada minimum staking?">
Ya. Jumlah minimum yang dapat Anda stake adalah 50 B3.
</Accordion>
</AccordionGroup>

<Warning>
Ingat bahwa periode pendinginan 45 hari adalah wajib untuk semua operasi unstaking. Rencanakan dengan baik jika Anda memerlukan akses ke token Anda pada tanggal tertentu.
</Warning>
