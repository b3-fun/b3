## Pertanyaan yang Sering Diajukan

<AccordionGroup>
<Accordion title="Bagaimana cara bergabung dengan turnamen?">
Pergi ke tab **Compete** di [**basement.fun**](https://basement.fun/), pilih turnamen yang sedang berlangsung, dan klik **Join**. Konfirmasi transaksi untuk mendaftar.
</Accordion>

<Accordion title="Bagaimana cara kerja taruhan?">
Anda dapat menantang teman atau membuka taruhan publik dengan mendanai taruhan di [basement.fun](https://basement.fun/). Pilih permainan apa saja, lalu klik ikon pedang untuk membuat tantangan taruhan.
</Accordion>

<Accordion title="Apa yang terjadi jika saya terputus selama pertandingan?">
Terputus dapat mengakibatkan kekalahan.
</Accordion>

<Accordion title="Mengapa permainan tidak dapat dimuat di situs?">
Pertama, pastikan koneksi internet Anda stabil. Jika masalah berlanjut, coba gunakan browser yang berbeda atau bersihkan cache Anda. Jika itu tidak membantu, hubungi dukungan untuk bantuan lebih lanjut.
</Accordion>

<Accordion title="Apakah saya bisa bermain di browser seluler?">
Beberapa permainan mendukung browser seluler, tetapi beberapa permainan hanya tersedia di desktop. Anda dapat memeriksa dukungan seluler di halaman detail permainan.
</Accordion>

<Accordion title="Apakah ada aplikasi seluler?">
Ya. Anda dapat mengunduh aplikasi BSMNT di Android dan Apple.
</Accordion>
</AccordionGroup>

<Tip>
Untuk pengalaman bermain game terbaik, pastikan Anda memiliki koneksi internet yang stabil dan menggunakan browser yang diperbarui. Chrome dan Firefox umumnya menyediakan kompatibilitas terbaik.
</Tip>

<Info>
Aplikasi seluler menyediakan pengalaman yang paling optimal untuk bermain game di seluler, jadi unduhlah dari toko aplikasi perangkat Anda untuk performa terbaik.
</Info>
