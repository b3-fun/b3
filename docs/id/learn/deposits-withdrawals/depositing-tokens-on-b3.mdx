Untuk berinteraksi dengan beberapa game dan fitur di ekosistem B3, Anda akan memerlukan token di dompet Anda. Panduan ini akan membantu Anda mengetahui cara menyetorkan token menggunakan jembatan B3.

## Cara Menyetorkan Token

<iframe
  className="w-full aspect-video rounded-xl"
  src="https://www.youtube.com/embed/-99zipMlOlQ"
  title="Cara menyetorkan token ke B3"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
></iframe>

<Steps>
<Step title="Log in ke basement.fun">
  Log in ke akun Anda di basement.fun.
</Step>

<Step title="Pilih jaringan dan token">
  Pilih jaringan dan token yang ingin Anda setorkan ke B3.
  
  <Frame>
    <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/685306b72e0550277aa9250e/images/687dfe81bb2dcf4c62e168fe/file-cwmhuEFxD0.png" alt="Antarmuka pemilihan jaringan dan token" />
  </Frame>
</Step>

<Step title="Masukkan jumlah dan jembatan">
  Masukkan jumlah token yang ingin Anda setorkan dan klik **Bridge**.
  
  <Frame>
    <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/685306b72e0550277aa9250e/images/687dff266828a66bbecf0bff/file-35Fdwt0jZC.png" alt="Input jumlah token dengan tombol Bridge" />
  </Frame>
</Step>

<Step title="Konfirmasi transaksi">
  Konfirmasi transaksi di dompet yang terhubung untuk menyelesaikan setoran Anda.
  
  <Frame>
    <img src="https://s3.amazonaws.com/helpscout.net/docs/assets/685306b72e0550277aa9250e/images/687e04626828a66bbecf0c0b/file-Cz781bdyhB.png" alt="Dialog konfirmasi dompet untuk transaksi setoran" />
  </Frame>
  
  <Check>
    Setelah transaksi selesai, token Anda akan muncul di dompet Anda.
  </Check>
</Step>
</Steps>

<Note>
Setoran biasanya selesai dalam beberapa menit, tetapi waktu pemrosesan dapat bervariasi tergantung pada kepadatan jaringan.
</Note>
