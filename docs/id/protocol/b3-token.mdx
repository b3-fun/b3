## Apa itu \$B3?

**\$B3** adalah token utilitas yang menggerakkan seluruh ekosistem B3. Sebagai token dasar untuk jaringan multichain yang berskala horizontal, B3 memungkinkan tata kelola, staking, dan nilai lintas appchains dan aplikasi yang dibangun di atas B3.
