## Native Bridge

<div style={{ marginBottom: '16px' }}>
  <span style={{ backgroundColor: '#3b82f6', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Base ke B3</span>
  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Penarikan 7 Hari</span>
  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>Tanpa Biaya</span>
</div>

Dengan mudah bridge dari Base ke B3 menggunakan official OP Stack canonical bridge!

<Warning>
Penarikan dari B3 kembali ke Base memiliki periode tantangan 7 hari untuk keamanan. Gunakan bridge pihak ketiga untuk penarikan instan.
</Warning>

<Steps>
<Step title="Kunjungi bridge">
  Pergi ke [bridge.b3.fun](https://bridge.b3.fun) untuk mengakses bridge resmi.
</Step>

<Step title="Sambungkan Dompet Anda">
  Sambungkan dompet Anda ke antarmuka bridge.
</Step>

<Step title="Pilih arah">
  Pilih apakah Anda ingin:
  - **Deposit**: Membawa dana dari Base → B3
  - **Penarikan**: Bridge kembali dari B3 → Base
</Step>

<Step title="Masukkan jumlah">
  Masukkan jumlah ETH yang ingin Anda bridge.
</Step>

<Step title="Selesaikan transaksi">
  Klik "Deposit" dan konfirmasi transaksi di dompet Anda.
  
  <Tip>
  Deposit ke B3 instan, tetapi penarikan membutuhkan waktu 7 hari untuk periode tantangan.
  </Tip>
</Step>
</Steps>

## Native Transfer (Bridge Contract)

<div style={{ marginBottom: '16px' }}>
  <span style={{ backgroundColor: '#3b82f6', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Deposit Instan</span>
  <span style={{ backgroundColor: '#f59e0b', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px', marginRight: '8px' }}>Penarikan 7 Hari</span>
  <span style={{ backgroundColor: '#10b981', color: 'white', padding: '4px 8px', borderRadius: '4px', fontSize: '12px' }}>Tanpa Biaya</span>
</div>

Kirim Base ETH melalui kontrak bridge. Transfer **Base ETH** ke alamat bridge berikut di **Base Mainnet**:

<Warning>
Metode ini hanya untuk pengguna lanjutan. Pastikan Anda mengirim dari Base Mainnet untuk menghindari kehilangan dana.
</Warning>

```text Bridge Address
0x769547a723783FCA36BAaf1ECcf9dfdbF6d09F38
```

<Note>
Ketika Anda mengirim ETH ke alamat ini dari Base, itu akan secara otomatis muncul di dompet Anda di B3. Ini adalah metode interaksi kontrak langsung.
</Note>

<Accordion title="Cara menggunakan kontrak bridge">
1. **Pastikan Anda di Base Mainnet** di dompet Anda
2. **Kirim ETH** ke alamat kontrak bridge di atas
3. **Tunggu konfirmasi** - dana Anda akan muncul di B3 secara otomatis
4. **Periksa dompet B3 Anda** - beralih ke jaringan B3 untuk melihat ETH yang di-bridge Anda

<Warning>
Periksa dua kali Anda berada di Base Mainnet sebelum mengirim. Mengirim dari jaringan lain dapat mengakibatkan kehilangan dana.
</Warning>
</Accordion>
