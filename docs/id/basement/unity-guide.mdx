## Contoh Kode

### Memicu Mesin Aturan

Berikut adalah contoh cara memicu mesin aturan dari dalam game Unity Anda:

```csharp Unity C# Example
B3LauncherClient.TriggerRulesEngine(new B3LauncherClient.TriggerRulesEngineBody
{
    launcherJwt = B3Instance.Instance.SessionJWT ?? jwtInput.text,
    trigger = "open-tip-modal",
    otherWallet = otherWalletInput.text,
}, null);
```

### Manajemen Sesi

SDK secara otomatis menangani sesi pengguna:

```csharp Session Example
// Akses JWT sesi saat ini
string sessionToken = B3Instance.Instance.SessionJWT;

// Periksa jika pengguna terotentikasi
if (B3Instance.Instance.IsAuthenticated)
{
    // Pengguna telah masuk dan siap berinteraksi dengan layanan B3
    Debug.Log("Pengguna terotentikasi dengan sukses");
}
```

## Integrasi API

Unity SDK menyediakan akses mulus ke semua API launcher B3:

<AccordionGroup>
<Accordion title="Panggilan Berbasis Callback">
  Pola callback tradisional untuk menangani respons API:
  
  ```csharp
  B3LauncherClient.SomeAPICall(requestData, (response) => {
      // Tangani respons
      Debug.Log("Panggilan API selesai");
  });
  ```
</Accordion>

<Accordion title="Panggilan Async/Await">
  Pola modern async/await untuk kode yang lebih bersih:
  
  ```csharp
  public async void CallAPI()
  {
      var response = await B3LauncherClient.SomeAPICallAsync(requestData);
      // Tangani respons
      Debug.Log("Panggilan API selesai");
  }
  ```
</Accordion>

<Accordion title="WebGL Webhooks">
  Menangani event launcher di build WebGL:
  
  ```csharp
  // Daftarkan penangan webhook
  B3Instance.Instance.OnWebhookReceived += HandleWebhook;
  
  private void HandleWebhook(WebhookData data)
  {
      // Proses data webhook masuk
      Debug.Log($"Webhook diterima: {data.type}");
  }
  ```
</Accordion>
</AccordionGroup>
