## Apa itu Basement?
Basement adalah lapisan yang berfokus pada game dari ekosistem B3, dirancang khusus untuk pengembang game dan komunitas gaming. Ini menyediakan:

<img src="/images/bsmnt-logo.svg" alt="Basement.fun" className="w-32" />

- **Game Launcher**: Menyebarkan dan mengelola game di B3
- **Alat Pengembang**: SDK dan API untuk pengembangan game
- **Fitur Komunitas**: Profil pemain, pencapaian, dan fitur sosial
- **Sumber Daya Merek**: Aset dan pedoman untuk branding yang konsisten

## Fitur Utama

<AccordionGroup>
<Accordion title="SDK Pengembangan Game">
Kit pengembangan perangkat lunak yang komprehensif untuk Unity, Construct, dan mesin game populer lainnya, memungkinkan integrasi mulus dengan fitur blockchain B3.
</Accordion>

<Accordion title="Identitas & Akun Pemain">
Sistem akun global yang memungkinkan pemain untuk mempertahankan profil, pencapaian, dan aset yang konsisten di semua game dalam ekosistem Basement.
</Accordion>

<Accordion title="Manajemen Aset">
Alat untuk mengelola aset dalam game, NFT, dan token, dengan integrasi mudah untuk pengembang dan kepemilikan yang transparan untuk pemain.
</Accordion>

<Accordion title="Fitur Komunitas">
Fitur sosial bawaan, papan peringkat, turnamen, dan alat manajemen komunitas untuk meningkatkan keterlibatan pemain.
</Accordion>
</AccordionGroup>
