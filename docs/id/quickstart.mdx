### Langkah 1: Pasang SDK B3
<Steps>
  <Step title="Pasang paket SDK">
    Pilih manajer paket pilihan Anda untuk memasang SDK B3:

    <CodeGroup>
    ```bash npm
    npm install @b3dotfun/sdk
    ```
    ```bash pnpm
    pnpm add @b3dotfun/sdk
    ```
    ```bash yarn
    yarn add @b3dotfun/sdk
    ```
    </CodeGroup>

    <Tip>SDK B3 sudah termasuk definisi TypeScript secara langsung!</Tip>
  </Step>

  <Step title="Impor gaya CSS">
    Impor gaya bawaan dalam aplikasi Anda:

    ```typescript
    import "@b3dotfun/sdk/index.css";
    ```
  </Step>
</Steps>

### Langkah 2: Tambahkan B3Provider
Bungkus B3Provider di sekitar akar aplikasi Anda:

```tsx
import { B3Provider, SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

function App() {
  return (
    <B3Provider environment="production">
      {children}
    </B3Provider>
  );
}
```


### Langkah 3: Gunakan SDK B3
<AccordionGroup>
 <Accordion icon="swap" title="Akun Instan">
    Aktifkan otentikasi akun global dengan B3:

    ```tsx
    import { SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

    function App() {
      return (
        <B3Provider environment="production">
          <SignInWithB3
            provider={{ strategy: "google" }}
            partnerId="your-partner-id"
            onLoginSuccess={(globalAccount) => {
              console.log("Authenticated:", globalAccount);
            }}
          />
        </B3Provider>
      );
    }
    ```
  </Accordion>
  <Accordion icon="swap" title="Akses Likuiditas">
    Aktifkan pertukaran lintas rantai & interaksi kustom dengan AnySpend:

    ```tsx
    import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

    function SwapPage() {
      return <AnySpend mode="page" />;
    }
    ```
  </Accordion>
</AccordionGroup>
