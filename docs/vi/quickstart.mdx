### Bước 1: Cài đặt B3 SDK
<Steps>
  <Step title="Cài đặt gói SDK">
    Chọn trình quản lý gói ưa thích của bạn để cài đặt B3 SDK:

    <CodeGroup>
    ```bash npm
    npm install @b3dotfun/sdk
    ```
    ```bash pnpm
    pnpm add @b3dotfun/sdk
    ```
    ```bash yarn
    yarn add @b3dotfun/sdk
    ```
    </CodeGroup>

    <Tip>B3 SDK bao gồm định nghĩa TypeScript ngay từ đầu!</Tip>
  </Step>

  <Step title="Nhập CSS styles">
    Nhập styles mặc định vào ứng dụng của bạn:

    ```typescript
    import "@b3dotfun/sdk/index.css";
    ```
  </Step>
</Steps>

### Bước 2: Thêm B3Provider
Bọc B3Provider quanh gốc của ứng dụng:

```tsx
import { B3Provider, SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

function App() {
  return (
    <B3Provider environment="production">
      {children}
    </B3Provider>
  );
}
```


### Bước 3: Sử dụng B3 SDK
<AccordionGroup>
 <Accordion icon="swap" title="Tài khoản tức thì">
    Kích hoạt xác thực tài khoản toàn cầu với B3:

    ```tsx
    import { SignInWithB3 } from "@b3dotfun/sdk/global-account/react";

    function App() {
      return (
        <B3Provider environment="production">
          <SignInWithB3
            provider={{ strategy: "google" }}
            partnerId="your-partner-id"
            onLoginSuccess={(globalAccount) => {
              console.log("Đã xác thực:", globalAccount);
            }}
          />
        </B3Provider>
      );
    }
    ```
  </Accordion>
  <Accordion icon="swap" title="Truy cập thanh khoản">
    Kích hoạt chuyển đổi chuỗi chéo & tương tác tùy chỉnh với AnySpend:

    ```tsx
    import { AnySpend } from "@b3dotfun/sdk/anyspend/react";

    function SwapPage() {
      return <AnySpend mode="page" />;
    }
    ```
  </Accordion>
</AccordionGroup>
