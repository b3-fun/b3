## Brand Overview

The B3 brand represents innovation, connectivity, and the future of blockchain technology. Our visual identity reflects the seamless integration of multiple blockchain networks into a unified ecosystem.

## Logo Usage

### Primary Logo

Use the primary B3 logo whenever possible for maximum brand recognition.

```html
<!-- SVG Logo -->
<img src="https://cdn.b3.fun/b3_logo.svg" alt="B3" width="120" height="40" />

<!-- PNG Logo (for non-SVG environments) -->
<img src="https://cdn.b3.fun/b3_logo.png" alt="B3" width="120" height="40" />
```

### Logo Don'ts

❌ Don't stretch or distort the logo
❌ Don't change the colors
❌ Don't add effects or shadows
❌ Don't place on busy backgrounds
❌ Don't use outdated versions

## Animation Guidelines

### Micro-interactions

```css
/* Hover Animations */
.hover-lift {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}

.hover-lift:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 25px rgba(0, 0, 0, 0.15);
}

/* Loading Animations */
@keyframes spin {
  to { transform: rotate(360deg); }
}

.spinner {
  animation: spin 1s linear infinite;
}

/* Fade Animations */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

.fade-in {
  animation: fadeIn 0.3s ease-out;
}
```

### Loading States

```css
.loading-skeleton {
  background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%);
  background-size: 200% 100%;
  animation: loading 1.5s infinite;
}

@keyframes loading {
  0% { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
```

## Brand Voice

### Tone and Messaging

- **Innovative**: Leading-edge technology and forward-thinking solutions
- **Accessible**: Complex blockchain technology made simple
- **Reliable**: Trustworthy infrastructure for developers and users
- **Community-focused**: Building together for the ecosystem
