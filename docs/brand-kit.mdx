## Brand Guidelines

<Steps>
<Step title="Let it breathe">
  **Maintain adequate spacing around the B3 logo**
  
  - Minimum clear space: 0.5x the logo height on all sides
  - Don't place text or other elements too close to the logo
  - Ensure the logo remains legible at all sizes
  
  <Tip>
  The logo should never appear smaller than 24px in height for digital applications
  </Tip>
</Step>

<Step title="Make it fun">
  **Keep the brand experience light, friendly, and engaging** - Use vibrant colors when appropriate - Include playful
  animations and micro-interactions - Write in a conversational, approachable tone - Embrace the gaming culture and
  community spirit
</Step>

<Step title="Stay consistent">
  **Use brand elements consistently across all touchpoints**
  
  - Stick to the approved color palette and fonts
  - Follow logo usage guidelines
  - Maintain the same tone and visual style
  - Reference this brand kit for all brand decisions
</Step>
</Steps>

## Motion & Animation

<Info>
  Animations are welcomed when representing B3 - after all, movement is fun! We recommend Framer Motion for React
  applications, but use whatever animation library works best for your stack.
</Info>

### Animation Principles

<AccordionGroup>
<Accordion title="Playful & Smooth">
  - Use ease-out transitions for natural feel
  - Add subtle bounce effects for gaming appeal
  - Keep animations under 500ms for responsiveness
</Accordion>

<Accordion title="Gaming-Inspired">
  - Include hover effects on interactive elements - Use scale transforms for button interactions - Add loading
  animations that feel game-like
</Accordion>

<Accordion title="Performance-First">
  - Use CSS transforms over position changes
  - Prefer opacity changes over visibility
  - Test animations on various device capabilities
</Accordion>
</AccordionGroup>
